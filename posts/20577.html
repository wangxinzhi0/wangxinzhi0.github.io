二叉树找最优节点（100，AC 85%）

题目描述

给出一颗二叉树，每个节点有一个编号和一个值，该值可能为负数，请你找出一个最优节点（除根节点外），使得该节点将树分成两棵树后（原来的树移除这个节点及其子节点，新的树以该节点为根节点），分成的两棵树各节点的和之间的差绝对值最大。请输出该节点编号，如有多个相同的差，输出编号最小的节点

输入描述

4
4 9 -7 -8
0 1
0 2
2 3

第一行一个正整数n>1表示二叉树的节点数，编号0~n-1。

第二行输入n个整数，表示每个节点的权值

剩余输入行代表节点指向，0 1代表0的左子树是1，0 2代表0的右子树是2，2 3代表2的左子树是3，注意，总是先出现左子树指向，然后才能出现右子树指向

输出描述

一行，节点编号，上述输入应该输出1

1

解题思路

  • 建立树节点，包含权值、编号、左右子树。根据输入节点指向构建二叉树。
  • 然后写一个求二叉树所有节点和的函数
  • 最后遍历编号1~n-1，找到相差最大的编号

AC不是100%好像是因为int爆了

import java.util.*;

public class Main {

        Scanner sc = new Scanner(System.in);
        //处理输入
        int n = sc.nextInt();//节点数
        long[] weights = new long[n];//权值
        for(int i=0; i<n; i++){
            weights[i] = sc.nextInt();
        }
        List<int[]> list = new ArrayList<>();//指向关系
        int a=0;
        while(a<3 && sc.hasNext()){
            list.add(new int[]{sc.nextInt(), sc.nextInt()});
            //a++;自测需要打开！！！
        }
        //构建二叉树
        TreeNode[] tree = new TreeNode[n];//创建节点数组
        for(int i=0; i<n; i++){
            tree[i] = new TreeNode(i, weights[i]);
        }
        for(int i=0; i<list.size(); i++){//指向修改
            int[] tmp = list.get(i);
            if(i==0){//第一个
                tree[tmp[0]].left = tree[tmp[1]];
            }else{//其他，根据和前一个是否相等判断左右子树
                if(list.get(i-1)[0]==tmp[0]){
                    tree[tmp[0]].right = tree[tmp[1]];
                }else{
                    tree[tmp[0]].left = tree[tmp[1]];
                }
            }
        }
        //遍历找到节点
        long max=0;
        long allSum = sum(tree[0]);//所有节点和
        int index=0;
        for(int i=1; i<n; i++){
            long tmp = Math.abs(2*allSum-sum(tree[i]));
            if(tmp>max){
                index = i;
                max = tmp;
            }
        }
        //输出
        System.out.println(index);
    }

    //求树的和
    private static long sum(TreeNode root){
        if(root==null) return 0;

        return root.val+sum(root.left)+sum(root.right);
    }
}

//节点

    int id;//编号
    long val;//权值
    TreeNode left, right;
    public TreeNode(int id, long val){
        this.id = id;
        this.val = val;
    }
}

走梅花桩阵（200，AC 47.62%）

题目描述

有一个M*N的梅花桩阵，每个桩都有允许跳跃的最大步数，用户从0,0的位置开始跳跃，允许向右和向下两个方向跳跃，求最少要跳跃多少次能到达M-1，N-1的位置。无法到达目的时返回-1

M<=100，N<=100，每个桩允许跳跃的最大步数均小于10的正整数，0表示不允许跳跃到该位置

输入描述

3,3 3 2 2 0 1 0 1 1 1

1、第一行为M和N，用,号隔开

2、第二行为M*N的梅花桩（格式参考样例），数组位置允许跳跃的最大步数，0表示该位置为空，不能跳跃到该位置

输出描述

一行，表示最少多少跳到M-1，N-1，不能到达返回-1

2

解题思路1

dfs暴力搜索所有跳跃结果，然后找最少跳跃次数

代码1

//运行超时
import java.util.*;

public class Main {
    private static List<List<int[]>> res;
    private static List<int[]> path;

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        //处理输入
        String[] str = sc.nextLine().split("\\,");
        int m = Integer.parseInt(str[0]);
        int n = Integer.parseInt(str[1]);
        int[][] arr = new int[m][n];
        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                arr[i][j] = sc.nextInt();
            }
        }
        //调用递归
        res = new ArrayList<>();
        path = new ArrayList<>();
        path.add(new int[]{0,0});//加入起点
        dfs(arr, 0, 0);
        //找到最小步数
        int min=Integer.MAX_VALUE;
        for(List<int[]> list : res){
            int tmp = list.size();
            if(tmp<min) min = tmp;
        }
        if(res.size()==0) System.out.println(-1);
        else System.out.println(min-1);
    }

    private static void dfs(int[][] arr, int x, int y){
        //边界条件
        if(x>=arr.length||x<0||y>=arr[0].length||y<0||arr[x][y]==0) return;
        //终止条件
        if(x==arr.length-1 && y==arr[0].length-1){
            res.add(new ArrayList<>(path));return;
        }

        //选择走下
        for(int i=1; i<=arr[x][y] && i+x<arr.length; i++){
            path.add(new int[]{x+i, y});
            dfs(arr, x+i, y);
            path.remove(path.size()-1);
        }

        //选择走右
        for(int i=1; i<=arr[x][y] && i+y<arr[0].length; i++){
            path.add(new int[]{x, y+i});
            dfs(arr, x, y+i);
            path.remove(path.size()-1);
        }
    }
}

解题思路2

动态规划，dp[i][j]表示到数组i,j位置的最少步数

dp初始化

  • dp[0][0]=0
  • 其他位置初始化为Integer.MAX_VALUE/2

状态转移，从左往右，从上往下更新

  • 如果arr[i][j]==0，无法从该位置走，dp不更新
  • 否则，可以从该位置向右或者向下走arr[i][j]步，如果没有越界，更新走到这些位置的步数。以向右走为例，dp[i][j+k]=Math.min(dp[i][j+k],dp[i][j]+1)

代码2

//动态规划，dp指的是跳到i,j位置的最少步数
public class HuaWei2_1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        //处理输入
        String[] str = sc.nextLine().split("\\,");
        int m = Integer.parseInt(str[0]);
        int n = Integer.parseInt(str[1]);
        int[][] arr = new int[m][n];
        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                arr[i][j] = sc.nextInt();
            }
        }
        //动态规划数组
        int[][] dp = new int[m][n];
        //初始化dp数组
        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                dp[i][j] = Integer.MAX_VALUE/2;
            }
        }
        dp[0][0]=0;
        //递推关系
        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                if(arr[i][j]==0){//当前位置为0，dp不更新
                    continue;
                }else{
                    for(int k=1; k<=arr[i][j]; k++){//右边，或下边arr[i-1][j-1]个位置可以由i,j处直接跳到
                        if(j+k<n && arr[i][j+k]!=0){//向右走
                            dp[i][j+k] = Math.min(dp[i][j+k], dp[i][j]+1);
                        }
                        if(i+k<m && arr[i+k][j]!=0){//向下走
                            dp[i+k][j] = Math.min(dp[i+k][j], dp[i][j]+1);
                        }
                    }
                }
            }
        }
        System.out.println(dp[m-1][n-1]==Integer.MAX_VALUE/2 ? -1 : dp[m-1][n-1]);
    }
}

HJ6 质数因子

题目地址：https://www.nowcoder.com/practice/196534628ca6490ebce2e336b47b3607?tpId=37&&tqId=21229&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：迭代

分解质因数，从最小的开始（2）不停做除法，除到为1。或者除到最后除不了，就输出自己。还可以判断下除数是不是质数，但没必要，还会增加判断次数，得不偿失。

每找到一个质因数，就不断除以它，直到除尽，这样保证找到的因数一定是质数。

「有两个坑」

  • 这个数本身就是个质数，输出自己。
  • 可能超时，和判断质数一样，没必要从2一直尝试到number，循环到Math.sqrt(number)即可。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        long number = sc.nextLong();
        //从2开始，判断到sqrt(number)即可
        for(int i=2; i<=Math.sqrt(number); i++){
            //每找到一个质因数，除尽为止，这样保证找到的因数一定是质数
            while(number%i==0){
                System.out.print(i+" ");
                number /= i;
            }
        }
        //最后number不为1，说明除完后剩余一个质数
        System.out.println(number==1 ? "":number+" ");
    }
}

HJ16 购物单

题目地址：https://www.nowcoder.com/practice/f9c6f980eeec43ef85be20755ddbeaf4?tpId=37&&tqId=21239&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：动态规划

这篇博客很生动的讲述了背包问题的动态规划思想：https://www.jianshu.com/p/a66d5ce49df5，看完博客，再来看题目就很好做，购物单问题实际跟博客里的背包问题差不多。

在购物问题里，先处理主件行数据，需要考虑一下四种情况：

  • 主件
  • 主件+附件1
  • 主件+附件2
  • 主件+附件1+附件2

主件行数据从以上四种情况里挑选出最优的一种情况，然后它的附件行只需要照抄主件行数据就行。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        //输入数据
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();//总钱数
        int m = sc.nextInt();//商品数
        Goods[] goods = new Goods[m+1];
        for(int i=1; i<=m; i++){
            int v = sc.nextInt();//价格
            int p = sc.nextInt();//重要程度
            int q = sc.nextInt();//附件
            goods[i] = new Goods(v, p, q);
            //如果商品是q的附件，默认到商品i时，商品q已经放入数组（即先出现主件，再出现附件）
            if(q>0){
                if(goods[q].addID1==0) goods[q].addID1=i;//商品q还没添加附件
                else goods[q].addID2=i;//商品q已经添加一个附件
            }
        }
        
        //处理网格
        int[][] dp = new int[m+1][n+1];
        for(int i=1; i<=m; i++){//第i个商品
            int v1=0, v2=0, v3=0, v4=0;
            int w1=0, w2=0, w3=0, w4=0;
            
            //1.主件
            v1 = goods[i].v;
            w1 = v1*goods[i].p;
            
            //2.主件+附件1
            if(goods[i].addID1!=0){
                v2 = v1 + goods[goods[i].addID1].v;
                w2 = w1 + goods[goods[i].addID1].v*goods[goods[i].addID1].p;
            }
            
            //3.主件+附件2
            if(goods[i].addID2!=0){
                v3 = v1 + goods[goods[i].addID2].v;
                w3 = w1 + goods[goods[i].addID2].v*goods[goods[i].addID2].p;
            }
            
            //4.主件+附件1+附件2
            if(goods[i].addID1!=0 && goods[i].addID2!=0){
                v4 = v2+v3-v1;
                w4 = w2+w3-w1;
            }
            
            for(int j=1; j<=n; j++){
                if(goods[i].q>0){
                    //附件行数据会和它的主件行数据一模一样，直接拷贝上一行数据即可
                    dp[i][j] = dp[i-1][j];
                }else{
                    //对于主件行数据而言，它需要考虑四种情况里，自己选择哪种能够利益最大化
                    dp[i][j] = dp[i-1][j];
                    if(v1<=j) dp[i][j] = Math.max(dp[i][j], w1+dp[i-1][j-v1]);
                    if(v2<=j) dp[i][j] = Math.max(dp[i][j], w2+dp[i-1][j-v2]);
                    if(v3<=j) dp[i][j] = Math.max(dp[i][j], w3+dp[i-1][j-v3]);
                    if(v4<=j) dp[i][j] = Math.max(dp[i][j], w4+dp[i-1][j-v4]);
                }
            }
        }
        //输出数据
        System.out.println(dp[m][n]);
    }
}

//商品类
class Goods{
    int v;
    int p;
    int q;
    
    int addID1=0;//附件1的ID
    int addID2=0;//附件2的ID
    
    public Goods(int v, int p, int q){
        this.v=v;
        this.p=p;
        this.q=q;
    }
}

HJ18 识别有效的IP地址和掩码并进行分类统计

方法一：模拟

注意事项

  • Java的split正则匹配加上前缀"\\"；
  • 掩码拼接成32位字符串，每个点分十进制位Integer.toBinaryString()不包含前导0，要拼接成8位

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int[] arr = new int[7];
        while(sc.hasNext()){
            String[] str = sc.nextLine().split("\\~");
            if(isValidIP(str[0]) && isValidMask(str[1])){
                //有效IP和掩码，分类统计
                int num = Integer.parseInt(str[0].split("\\.")[0]);
                if(num>=1 && num<=126) arr[0]++;
                if(num>=128 && num<=191) arr[1]++;
                if(num>=192 && num<=223) arr[2]++;
                if(num>=224 && num<=239) arr[3]++;
                if(num>=240 && num<=255) arr[4]++;
                
                int num2 = Integer.parseInt(str[0].split("\\.")[1]);
                if(num==10) arr[6]++;
                if(num==172 && (num2>=16&&num2<=31)) arr[6]++;
                if(num==192 && num2==168) arr[6]++;
            }else{//错误IP或掩码
                arr[5]++;
            }
        }
        //输出结果
        for(int i=0; i<7; i++){
            if(i!=6) System.out.print(arr[i]+" ");
            else System.out.print(arr[i]);
        }
    }
    
    //是否有效IP
    private static boolean isValidIP(String ip){
        String[] str = ip.split("\\.");
        //长度是否正确
        if(str.length!=4) return false;
        
        //数字范围是否正确
        int num=0;
        for(int i=0; i<4; i++){
            num = Integer.parseInt(str[i]);
            if(num<0 || num>255) return false;
        }
        return true;
    }
    
    //是否有效掩码
    private static boolean isValidMask(String mask){
        //掩码拼接成32位字符串
        String[] str = mask.split("\\.");
        StringBuilder sb = new StringBuilder();
        for(int i=0; i<4; i++){
            String temp = Integer.toBinaryString(Integer.parseInt(str[i]));
            for(int j=0; j<8-temp.length(); j++){
                temp = "0"+temp;
            }
            sb.append(temp);
        }
        //全1全0情况
        if(sb.indexOf("0")==-1 || sb.indexOf("1")==-1) return false;
        
        if(sb.indexOf("0")<sb.lastIndexOf("1")) return false;
        return true;
    }
}

用LinkedHashMap记录错误，保证迭代输出顺序，输出最后8条错误，错误不满8条则全部输出。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);

            }
        }
    }
}

HJ20 密码验证合格程序

题目地址：https://www.nowcoder.com/practice/184edec193864f0985ad2684fbc86841?tpId=37&&tqId=21243&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：字符串

关键是第三条，不能有相同长度大于2的子串重复。从0~keys.length()-4取3位，看取位置后一位开始的剩余子串是否包含3位重复（有3位重复就是有相同长度大于2的子串重复）。

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            String keys = sc.nextLine();
            if(keys.length()<=8){//长度<=8
                System.out.println("NG");
            }else{
                if(check2(keys) && check3(keys)){
                    System.out.println("OK");
                }else{
                    System.out.println("NG");
                }
            }
            
        }
    }
    
    //要求2判断
    private static boolean check2(String keys){
        int a=0, b=0, c=0, d=0;
        for(int i=0; i<keys.length(); i++){
            char ch = keys.charAt(i);
            if(ch>='A' && ch<='Z') a=1;
            else if(ch>='a' && ch<='z') b=1;
            else if(ch>='0' && ch<='9') c=1;
            else d=1;
        }
        if(a+b+c+d>=3) return true;
        else return false;
    }
    
    //关键是第三点的判断，巧妙的用了substring和contains
    private static boolean check3(String keys){
        for(int i=0; i<keys.length()-3; i++){
            if(keys.substring(i+1).contains(keys.substring(i, i+3))) return false;
        }
        return true;
    }
}

HJ24 合唱队

方法一：动态规划

最长子序列问题

  • **先找到每个位置i左侧的最长上升子序列长度numL[i]**：等于左侧比它小的所有位置的最长子序列的最大值+1
  • **再找到每个位置i右侧的最长下降子序列长度numR[i]**：等于右侧比它小的所有位置的最长子序列的最大值+1
  • 然后求出所有位置的最长序列长度：等于左侧最长子序列长度+右侧最长子序列长度-1（因为该位置被算了两次，所以减1）
  • 最后用数组长度减去最长序列长度就是需要出队的同学数

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] height = new int[n];
            for(int i=0; i<n; i++){
                height[i] = sc.nextInt();
            }
            //每个位置左侧最长上升子序列
            int[] numL = new int[n];
            for(int i=0; i<n; i++){
                for(int j=0; j<i; j++){//左侧比height[i]小的位置最长上升序列长度
                    if(height[i]>height[j]) numL[i] = Math.max(numL[i], numL[j]);
                }
                numL[i] += 1;//左侧比height[i]小位置最长上升序列长度+1
            }
            //每个位置右侧最长下降子序列
            int[] numR = new int[n];
            for(int i=n-1; i>=0; i--){
                for(int j=n-1; j>i; j--){//右侧比heigth[i]小的位置最长下降序列长度
                    if(height[i]>height[j]) numR[i] = Math.max(numR[i], numR[j]);
                }
                numR[i] += 1;//右侧比height[i]小位置最长下降序列长度+1
            }
            //根据每个位置最长合唱队numL[i]+numR[i]-1，求出所有位置中最大值
            int maxLen=0;
            for(int i=0; i<n; i++){
                if(numL[i]+numR[i]-1>maxLen) maxLen = numL[i]+numR[i]-1;
            }
            //最后n-maxLen即为需要出列人数
            System.out.println(n-maxLen);
        }
    }
}

HJ28 素数伴侣

题目地址：https://www.nowcoder.com/practice/b9eae162e02f4f928eac37d7699b352e?tpId=37&&tqId=21251&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：匈牙利算法

素数除了2，其他是奇数，题目数组取值范围[2,30000]，两数的和不可能为2，所以只考虑奇数素数。要得到奇数，只能是奇数+偶数，这就得到一个二分图，对二分图做最大匹配，得到最多的匹配对数，有一个经典的算法，匈牙利算法。

二分图：你可以把图上点分成两堆，每堆之间不会有边，两个堆之间才可能连边。本题分成奇数和偶数两堆，刚好是个二分图问题。

代码

import java.util.*;

public class Main{
    //[2,30000]范围，素数不是偶数，和是素数只能是奇数+偶数，二分图最大匹配，匈牙利算法
    //匈牙利算法，先到先得 能让就让。有机会上，没机会创造机会也要上
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] arr = new int[n];
            for(int i = 0; i < n; i++){
                arr[i] = sc.nextInt();
            }
            //分两组，奇/偶
            ArrayList<Integer> evens = new ArrayList<Integer>();
            ArrayList<Integer> odds = new ArrayList<Integer>();
            for(int i = 0; i < n; i++){
                if((arr[i] & 1) != 1){
                    evens.add(arr[i]);
                }else{
                    odds.add(arr[i]);
                }
            }
            //下面开始才是重头戏
            //当前下标偶数的奇数匹配
            int[] evensMatch = new int[evens.size()];
            int result = 0;
            //遍历奇数去匹配偶数
            for(int i = 0; i < odds.size(); i++){
                //每一步重新创建，也就是相当于清空，boolean数组用于标记某个某数位置是否已匹配
                boolean[] isVisited = new boolean[evens.size()];
                //这里采用了匈牙利算法，先到先得
                if(find(odds.get(i), evens, isVisited, evensMatch)){
                    result++;
                }
            }
            System.out.println(result);
        }
    }
    
    public static boolean isPrime(int num){
        for(int i = 2; i * i <= num; i++){
            if(num % i == 0) {
                return false;
            }
        }
        return true;
    }
    
    public static boolean find(int x, ArrayList<Integer> evens, boolean[] isVisited, int[] evensMatch){
        //遍历偶数，去检查当前传入的奇数能否与偶数哪些数匹配
        for(int i = 0; i < evens.size(); i++){
            //如果当前偶数与传入的奇数匹配，并且当前偶数位还没有匹配过奇数
            if(isPrime(x + evens.get(i)) && !isVisited[i]){
                //设置当前偶数位匹配为true，也就是 1
                isVisited[i] = true;
                //如果第i个偶数没有伴侣，或者第i个偶数原来有伴侣，并且该伴侣能够重新找到伴侣的话(这里有递归调用)
                //则奇数x可以设置为第i个偶数的伴侣
                //这里采用了匈牙利算法，能让则让 
                if(evensMatch[i] == 0 || find(evensMatch[i], evens, isVisited, evensMatch)) {
                    evensMatch[i] = x;
                    return true;
                }
            }
        }
        //遍历完偶数都没有可以与传入奇数做伴侣的，该奇数只能孤独终老了
        return false;
    }
}

HJ41 称砝码

题目地址：https://www.nowcoder.com/practice/f9a4c19050fc477e9e27eb75f3bfd49c?tpId=37&&tqId=21264&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：哈希集合

将所有砝码放入集合。设置一个去重集合HashSet，先放入0，然后遍历砝码集合，每遍历一个砝码就拿HashSet中现有元素与该砝码值相加，得到新增加的重量集合，然后把新增加的重量再放入HashSet集合中，遍历结束HashSet的大小就是可以称的重量数目。

例如有2个重量1g和2g的砝码

  • 当遍历第一个砝码{0}->{0, 0+1}->{0, 1}

  • 当遍历第二个砝码{0, 1}->{0, 1, 0+2, 1+2}->{0, 1, 2, 3}

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] weights = new int[n];
            int[] counts = new int[n];
            for(int i=0; i<n; i++){
                weights[i] = sc.nextInt();
            }
            for(int i=0; i<n; i++){
                counts[i] = sc.nextInt();
            }
            //将所有砝码放入集合中
            List<Integer> list = new ArrayList<>();
            for(int i=0; i<n; i++){
                for(int j=0; j<counts[i]; j++){
                    list.add(weights[i]);
                }
            }
            //计算不同重量数
            int sum = countWeights(list);
            System.out.println(sum);
        }
    }
    
    private static int countWeights(List<Integer> list){
        //重量集合，先放入0
        Set<Integer> weights = new HashSet<>();
        weights.add(0);
        //遍历砝码，每遍历一个就与现有重量集合所有元素相加，得到新增加的重量
        for(int i=0; i<list.size(); i++){
            Set<Integer> newWeigths = new HashSet<>();
            for(int j : weights){//得到新增加的重量集合
                newWeigths.add(j + list.get(i));
            }
            for(int j : newWeigths){//新增加的重量放入总集合中
                weights.add(j);
            }
        }
        return weights.size();
    }
}

方法一：深度优先搜索

典型的dfs，设置好结束条件和剪枝条件，选择和撤销选择。注意点是，选择后将网格值修改为障碍1，撤销选择后修改为可走0，能够节省一个isVisited布尔数组。

//回溯
import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            int n = sc.nextInt();
            int m = sc.nextInt();
            int[][] maze = new int[n][m];
            for(int i=0; i<n; i++){
                for(int j=0; j<m; j++){
                    maze[i][j] = sc.nextInt();
                }
            }
            //深度优先搜索
            List<List<int[]>> ans = new ArrayList<>();
            List<int[]> path = new ArrayList<>();
            dfs(maze, 0, 0, path, ans);
            //题目保证了唯一，这里还是找一下最短路径
            int minPath = Integer.MAX_VALUE;
            int minIndex = 0;
            for(int i=0; i<ans.size(); i++){
                int size = ans.get(i).size();
                if(minPath>size){
                    minIndex=i;
                    minPath = size;
                }
            }
            //输出最短路径
            for(int[] arr : ans.get(minIndex)){
                System.out.println("("+arr[0]+","+arr[1]+")");
            }
        }
    }
    
    private static void dfs(int[][] maze, int i, int j, List<int[]> path, List<List<int[]>> ans){
        //结束条件，一定要用路径新建集合，path最后会remove为空
        if(i==maze.length-1 && j==maze[0].length-1){
            path.add(new int[]{i, j});
            ans.add(new ArrayList<>(path));
            return;
        }
        //边界条件，剪枝
        if(i<0 || i>=maze.length || j<0 || j>=maze[0].length || maze[i][j]==1) return;
        
        //i,j加入集合,并且当前位置设置为障碍
        path.add(new int[]{i, j});
        maze[i][j]=1;
        //向四个方向递归
        dfs(maze, i+1, j, path, ans);
        dfs(maze, i, j+1, path, ans);
        dfs(maze, i-1, j, path, ans);
        dfs(maze, i, j-1, path, ans);
        //找到死胡同，回溯，撤销i,j选择，当前位置设置为可走
        path.remove(path.size()-1);
        maze[i][j]=0;
    }
}

方法一：深度优先搜索

大致步骤如下

  • 从(0,0)开始搜索dfs(arr, 0, 0)，策略是从左到右，从上到下

    • 如果搜索行等于9，结束，返回true
    • 如果搜索列等于9，返回dfs(arr, row+1, 0)
    • 如果当前位置不为0，搜索下一列，dfs(arr, row, col+1)

  • 如果不满足上述条件，则是值为0的格子，从1-9尝试填入，判断填入是否满足同行同列同九宫格条件，同时递归其它0格子填充时不冲突

    • 同时满足，返回true，否则，撤销选择arr[row][col]=0，尝试其他填充

  • 如果1-9填充都不行，返回false

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //初始化数独
            int[][] arr = new int[9][9];
            for(int i=0; i<9; i++){
                for(int j=0; j<9; j++){
                    arr[i][j] = sc.nextInt();
                }
            }
            //调用处理函数
            dfs(arr, 0, 0);
            //输出结果
            for(int i=0; i<9; i++){
                for(int j=0; j<9; j++){
                    System.out.print(arr[i][j]);
                    if(j!=8){
                        System.out.print(" ");
                    }
                }
                System.out.println();
            }
        }
    }
    
    //dfs
    private static boolean dfs(int[][] arr, int row, int col){
        //终止和剪枝
        if(row==9) return true;
        if(col==9) return dfs(arr, row+1, 0);
        if(arr[row][col]!=0) return dfs(arr, row, col+1);
        
        //尝试填充1-9
        for(int i=1; i<=9; i++){
            //选择i填充
            arr[row][col] = i;
            //是否满足同行同列通九宫格条件、递归处理其他位置
            if(check(arr, row, col) && dfs(arr, row, col+1)){
                return true;
            }
            //撤销选择，回溯
            arr[row][col] = 0;
        }
        return false;
    }
    
    //判断填入数字的同行同列同九宫格是否符合条件
    private static boolean check(int[][] arr, int row, int col){
        //同行
        for(int i=0; i<9; i++){
            if(i!=col && arr[row][i]==arr[row][col]) return false;
        }
        
        //同列
        for(int i=0; i<9; i++){
            if(i!=row && arr[i][col]==arr[row][col]) return false;
        }
        
        //同九宫格
        int up = row/3*3;
        int left = col/3*3;
        for(int i=up; i<up+3; i++){
            for(int j=left; j<left+3; j++){
                if(i!=row && j!=col && arr[i][j]==arr[row][col]) return false;
            }
        }
        
        return true;
    }
}

        if(cur.next!=null) cur.next = cur.next.next;
        return head;
    }

HJ52 计算字符串的距离

题目地址：https://www.nowcoder.com/practice/3959837097c7413a961a135d7104c314?tpId=37&&tqId=21275&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：动态规划

看到字符串修改代价第一时间想到动态规划

传递公式

  • dp[i][j]表示字符串a的[1...i]和字符串b的[1...j]的字符串距离
  • 插入和删除互为逆过程，a删除指定字符变b，等同于b插入指定字符
  • 如果a[i]==b[j]，则说明a[i]和b[j]分别加入a，b之后不会影响字符串距离，dp[i][j]=dp[i-1][j-1]
  • 如果a[i]!=b[j]，则需要考虑三种可能，取最小值
    • a中插入字符，即dp[i][j]=dp[i-1][j]+1
    • b中插入字符，即dp[i][j]=dp[i][j-1]+1
    • a[i]替换成b[j]，即dp[i][j]=dp[i-1][j-1]+1

初始条件

  • 第一行，a为空，初始值等于列数
  • 第一列，b为空，初始值等于行数

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //调用距离计算函数
            int res = getDistance(sc.next(), sc.next());
            System.out.println(res);
        }
    }
    
    private static int getDistance(String str1, String str2){
        //特例
        if(str1.length()==0) return str2.length();
        if(str2.length()==0) return str1.length();
        
        //动态规划矩阵
        int len1=str1.length(), len2=str2.length();
        int[][] dp = new int[len1+1][len2+1];
        //初始值，首行首列
        for(int i=0; i<=len2; i++){
            dp[0][i] = i;
        }
        for(int i=0; i<=len1; i++){
            dp[i][0] = i;
        }
        //逐行推出dp[i][j]
        for(int i=1; i<=len1; i++){
            for(int j=1; j<=len2; j++){
                if(str1.charAt(i-1)==str2.charAt(j-1)){//相等，拷贝上一行前一列
                    dp[i][j]=dp[i-1][j-1];
                }else{//不相等，分三种情况，a中插入字符，b中插入字符，a[i]替换成b[j]
                    dp[i][j]=Math.min(Math.min(dp[i-1][j], dp[i][j-1]), dp[i-1][j-1]) + 1;
                }
            }
        }
        //返回字符串距离
        return dp[len1][len2];
    }
}

方法一：递归

令(m,n)表示将m个苹果放入n个盘子中的摆放方法总数

  • 当n>m时，必定有n-m个盘子永远空着，去掉它们对结果无影响，只需求(m,m)

  • 当n<=m时，放苹果分为两种情况，一种是有盘子为空，一种是每个盘子上都有苹果。

    • 假设有一个盘子为空，则(m,n)问题转化为m个苹果放到n-1个盘子上，即求(m,n-1)即可
    • 假设所有盘子都装有苹果，则盘子上至少有一个苹果，即最多剩下m-n个苹果，问题转化为m-n个苹果放到n个盘子上，即求(m-n, n)

难点理解，为什么是一个盘子而不是至少一个呢？因为至少一个包含在内，递归实现！

终止条件，递归两条路一条m会减少，一条n会减少，当苹果为0或盘子为1，返回1。

方法一：深度优先搜索

类似全排列的深度优先搜索，使用任意一个数字作为起始值，和一个标识数字是否已访问布尔数组，遍历数字进行递归和回溯即可

终止条件，数字使用完且计算结果为24，返回true

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        while(in.hasNext()){
            //处理输入
            String[] a = in.nextLine().split(" ");
            int[] nums = new int[4];
            for(int i = 0; i < 4; i ++){
                nums[i] = Integer.parseInt(a[i]);
            }
            //isVisited标识是否访问过
            boolean[] isVisited = new boolean[4];
            //以任意一个数作为开始
            boolean res = false;
            for(int i=0; i<4; i++){
                isVisited[i]=true;
                if(dfs(nums, 1, nums[i], isVisited)){
                    res=true;
                    break;
                }
                //以i开始无法得到24，撤销访问，尝试下一个数字开始
                isVisited[i]=false;
            }
            //输出结果
            System.out.println(res);
        }
    }

    private static boolean dfs(int[] nums, int counts, double val, boolean[] isVisited){
        //终止条件
        if(val == 24 && counts == 4) return true;
        
        //递归
        for(int i = 0; i < 4; i ++){
            if(!isVisited[i]){//未使用数字
                //选择数字使用
                isVisited[i] = true;
                if(dfs(nums, counts+1, val+nums[i], isVisited) || //加
                   dfs(nums, counts+1, val-nums[i], isVisited) || //减
                   dfs(nums, counts+1, val*nums[i], isVisited) || //乘
                   dfs(nums, counts+1, val/nums[i], isVisited))   //除
                      return  true;
                //撤销数字选择
                isVisited[i] = false;
            }
        }
        return false;
    }
}

HJ71 字符串通配符

题目地址：https://www.nowcoder.com/practice/43072d50a6eb44d2a6c816a283b02036?tpId=37&&tqId=21294&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：递归

采用递归思路，从前向后依次匹配

  • 遇到相同字符，不区分大小写，都向后移动一个字符
  • **遇到通配符?**，匹配一个字符（字母，数字或.），都向后移动一个字符
  • **遇到通配符***，三种情况，匹配（0个，1个，任意多个字符）
    • 匹配0个，通配符向后移动一个字符，字符串不动
    • 匹配1个，都向后移动一个字符
    • 匹配多个，通配符不动，字符串向后移动一个字符

递归的终止条件

  • 如果通配符和字符串同时到达末尾，返回true
  • 如果通配符或字符串其中一个到达末尾（不同时），返回false

注意事项

  • **连续的***，跳过之前的，只处理最后一个，否则会超时！
  • *和?只能匹配字母，数字或.

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            boolean res = dfs(sc.nextLine(), sc.nextLine(), 0, 0);
            System.out.println(res);
        }
    }
    
    private static boolean dfs(String s1, String s2, int p1, int p2){
        //终止条件
        if(p1==s1.length() && p2==s2.length()) return true;
        else if(p1==s1.length() || p2==s2.length()) return false;
        
        //遇到字符相等，不区分大小写
        if(Character.toLowerCase(s1.charAt(p1))==Character.toLowerCase(s2.charAt(p2))){
            //均后移
            return dfs(s1, s2, p1+1, p2+1);
        }else if(s1.charAt(p1)=='?'){//遇到字符?
            //如果匹配的不是字符数字或.，返回false
            if(!(Character.isLetterOrDigit(s2.charAt(p2)) || s2.charAt(p2)=='.')) return false;
            //否则，均后移一位
            return dfs(s1, s2, p1+1, p2+1);
        }else if(s1.charAt(p1)=='*'){//遇到字符*
            //连续只处理最后一个*，否则会超时
            while(p1+1<s1.length() && s1.charAt(p1+1)=='*') p1++;
            //如果p2指向不是能匹配字符，*匹配0个字符，p1后移
            if(!(Character.isLetterOrDigit(s2.charAt(p2)) || s2.charAt(p2)=='.')){
                return dfs(s1, s2, p1+1, p2);
            }
            //否则，p2指向字符可以匹配，*可以匹配0个，1个，或多个
            return dfs(s1, s2, p1+1, p2) || dfs(s1, s2, p1+1, p2+1) || dfs(s1, s2, p1, p2+1);
        }
        //其他不能匹配情况
        return false;
    }
}

HJ77 火车进站

题目地址：https://www.nowcoder.com/practice/97ba57c35e9f4749826dc3befaeae109?tpId=37&&tqId=21300&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：回溯

所有火车进出站都是由多步操作组合完成，而每步操作只有两种可能，要么只进站，要么只出站，然后继续递归，递归完成后恢复操作前的现场即可。

终止条件

  • 已全出站，out栈大小等于火车数组arr长度，注意new ArrayList<>(out)是从栈顶到栈底的元素顺序生成，而答案出站顺序是栈底到栈顶，结果转为字符串要从后往前遍历list，方便后续排序。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] arr = new int[n];
            for(int i=0; i<arr.length; i++) arr[i] = sc.nextInt();
            //双栈和Set集合
            Deque<Integer> in = new LinkedList<>();//站内的车
            Deque<Integer> out = new LinkedList<>();//已经出站的车
            List<String> res = new ArrayList<>();//结果集合
            //调用函数
            dfs(arr, 0, in, out, res);
            //排序
            Collections.sort(res);
            //输出结果
            for(String str : res){
                System.out.println(str);
            }
        }
    }
    
    //arr火车数组，k当前arr位置，in/out入站栈和出栈站，res结果
    private static void dfs(int[] arr, int k, Deque<Integer> in, Deque<Integer> out, List<String> res){
        //已全部出站
        if(out.size()==arr.length){
            //注意！！！new ArrayList<>()处理栈时是从栈顶到栈底的元素顺序
            List<Integer> list = new ArrayList<>(out);
            StringBuilder sb = new StringBuilder();
            for(int i=list.size()-1; i>=0; i--){
                if(i!=0) sb.append(list.get(i)+" ");
                else sb.append(list.get(i));
            }
            res.add(sb.toString());
            return;
        }
        
        if(!in.isEmpty()){//只出站不进站
            //选择出站
            int temp = in.peek();
            out.push(in.pop());
            dfs(arr, k, in, out, res);//出站不影响k
            //撤销选择，恢复现场
            in.push(temp);
            out.pop();
        }
        
        if(k!=arr.length){//只进站，不出站
            //选择进站
            in.push(arr[k]);
            dfs(arr, k+1, in, out, res);
            //撤销选择
            in.pop();
        }
    }
}

方法一：深度优先搜索

先把三和五的倍数都挑出来，算好两边的和sum3和sum5，所有数总和为sum，不是3或5倍数的剩余的数放在集合中。
求出target = sum/2-sum3或者target=sum/2-sum5作为目标数，看list中找能不能凑出target。
在剩余集合中找target是一个dfs过程

  • 终止条件，用完集合数，返回target==0
  • 两种递归情况
    • 选择start位置，递归找新目标数target-list.get(start)
    • 不选择start位置，在start+1和以后位置找target

特例：sum不是2的倍数，不可等分成两份，直接输出false

代码

import java.util.*;
public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        while(in.hasNext()){
            //根据输入计算sum3，sum5和所有数总和sum，同时把不是5和3倍数的剩余数放入集合
            List<Integer> list = new LinkedList<>();
            int n = in.nextInt();
            int sum5=0, sum3=0, sum = 0;
            for (int i = 0; i < n; i++){
                int cur = in.nextInt();//输入
                if (cur % 5 == 0){//5倍数和
                    sum5 += cur;
                }else if (cur % 3 == 0){//3倍数和
                    sum3 += cur;
                }else{//剩余加入集合
                    list.add(cur);
                }
                sum += cur;//总和
            }
            
            //特例，总和不是2的倍数，不可分2份和相等的数字
            if(sum%2!=0) System.out.println("false");
            else{//否则，在剩余数中找目标数字
                int target = sum/2 - sum3;//也可以是sum/2-sum5
                System.out.println(helper(list, target, 0));
            }
        }
    }

    private static boolean helper(List<Integer> list, int target, int start){
        if (start == list.size()) return target == 0;//终止条件
        
        //选择start位置，递归找新目标数target-list.get(start)， 或者不选择start位置，在后面位置找target
        return helper(list, target-list.get(start), start+1) || helper(list, target, start+1);
    }
}

方法一：List集合定制排序

由于Map集合定制排序比较困难，所以自己建立一个Node节点，存储key-value，将统计结果放入List集合，定制排序实现Comparator接口即可。

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            String s = sc.nextLine();
            //统计字符
            int[] arr = new int[36];
            for(int i=0; i<s.length(); i++){
                char c=s.charAt(i);
                if(c>='a' &&c<='z'){//字母
                    arr[c-'a']++;
                }else{//数字
                    arr[c-'0'+26]++;
                }
            }
            //统计结果存入List集合
            List<Node> list = new ArrayList<>();
            for(int i=0; i<36; i++){
                if(arr[i]!=0){
                    if(i<26){//字母
                        list.add(new Node((char)('a'+i), arr[i]));
                    }else{//数字
                        list.add(new Node((char)('0'+i-26), arr[i]));
                    }
                }
            }
            //排序
            Collections.sort(list, new Comparator<Node>(){
                public int compare(Node a, Node b){
                    if(a.v!=b.v){//根据值排序
                        return b.v-a.v;
                    }else{//值相等根据ASCII码排序
                        return a.k-b.k;
                    }
                }
            });
            //输出结果
            for(int i=0; i<list.size(); i++){
                System.out.print(list.get(i).k);
            }
            System.out.println();
        }
    }
}

    char k;
    int v;

        this.k=k;
        this.v=v;
    }

方法一：双指针+哈希集合

如果数组长度小于2，直接返回数组长度。

否则，初始化化指针[left,right)，将首元素加入set集合中，设置max=1。接着不断向右移动right指针

  • 如果set已经包含arr[right]，则删除左指针元素，并left++
  • 否则将right处元素加入集合，并right++
  • 做完指针移动之后，更新max=right-left

right移动到数组末尾后，返回max即为结果

代码

import java.util.*;

public class Solution {
    public int maxLength (int[] arr) {
        if(arr.length<2) return arr.length;
        
        Set<Integer> set = new HashSet<>();
        int left=0, right=1;//[left,right)，加入首元素
        int max = 1;
        set.add(arr[0]);
        while(right<arr.length){
            if(set.contains(arr[right])){//有重复
                set.remove(arr[left]);
                left++;
            }else{
                set.add(arr[right]);
                right++;
            }
            //更新max
            if(right-left>max) max = right-left;
        }
        return max;
    }
}

方法一：快排+二分

利用快排把数组分为两部分，一部分小于一个值，另一部分大于这个值的特点，进行二分查找

  • 以左端点为基准值，快排找到基准值的排序后位置i
    • 如果i==n-k，说明找到第K大，返回a[i]即可
    • 如果i<n-k，说明第K大在i的右边，递归到右边查找
    • 如果i>n-k，说明第K大在i的左边，递归到左边查找

代码

import java.util.*;

public class Solution {
    public int findKth(int[] a, int n, int K) {
        return quickSort(a, 0, n-1, K);
    }
    
    private int quickSort(int[] a, int left, int right, int K){
        //左端点为基值快排
        int base = a[left];
        int i=left, j=right;
        while(i<j){
            while(i<j && a[j]>=base) j--;
            while(i<j && a[i]<=base) i++;
            if(i<j){
                int temp = a[i];
                a[i] = a[j];
                a[j] = temp;
            }
        }
        a[left] = a[i];
        a[i] = base;
        
        if(i==a.length-K){//找到第K大
            return a[i];
        }else if(i<a.length-K){//i位置比第K大小，到右边找
            return quickSort(a, i+1, right, K);
        }else{//i位置比第K大大，到左边找
            return quickSort(a, left, i-1, K);
        }
    }
}

方法一：二分+动态规划

这道题目要求出最长递增子序列，而不是长度，而且设置了nlogn的时间复杂度。

如果暴力的话，只需要知道最大子序列长度和以下标i元素为结尾的子序列最大长度。但是这种办法在dp时只能暴力，如果想二分查找，必须得直到每个长度的序列里最后一个元素的最小值。因此需要维护两个数组和一个len

import java.util.*;

public class Solution {
    public int[] LIS (int[] arr) {
        int n = arr.length;
        //tails[k]表示长度为k+1的最长递增子序列的末尾元素
        int[] tails = new int[n];
        tails[0] = arr[0];
        int end = 0;
        //记录以当前元素为结尾的最长递增子序列的长度
        int[] maxLen = new int[n];
        Arrays.fill(maxLen, 1);

        for(int i=1; i<arr.length; i++){
            int left = 0;
            int right = end+1;

            while (left<right) {
                int mid=left + (right-left)/2;
                if (arr[i]>tails[mid]){
                    left = mid+1;
                } else {
                    right = mid;
                }
            }
            tails[left] = arr[i];
            if(left==end+1) end++;
            if(left+1 > maxLen[i]) maxLen[i] = left+1;
        }

        int[] res = new int[end+1];
        for (int i=n-1, j=res.length; j>0; i--) {
            if (maxLen[i]==j){
                res[--j] = arr[i];
            }
        }

        return res;
    }
}

方法一：哈希+双向链表

定义节点Node(k, v)作为双向链表的节点，用于表示最近最久未使用。定义Map<key,Node>存储和获取节点，数据结构如下图所示。

查询操作

  • 能找到则返回val，更新缓存（双向链表），否则返回-1

存入操作

  • 若缓存满，先处理缓存，删除链表尾节点，否则直接进行第二步
  • 然后加入新节点，更新缓存（双向链表）

缓存更新方法

  • 首先删除该节点，然后把节点插入到头部
  • 删除节点要判断节点是否已在缓存中，node.pre!=null才需要删除，否则是新节点，直接加入头部即可

代码

import java.util.*;

public class Solution {
    private Map<Integer, ListNode> map;
    private ListNode head, tail;
    
    public int[] LRU (int[][] operators, int k) {
        //初始化变量
        map = new HashMap<>();
        head = new ListNode(0, 0);
        tail = new ListNode(0, 0);
        head.next=tail;
        tail.pre=head;
        //遍历操作更新缓存，同时用List保存查询结果
        List<Integer> list = new ArrayList<>();
        for(int i=0; i<operators.length; i++){
            if(operators[i][0]==1){//set(key,value)放入元素
                int key = operators[i][1];
                int val = operators[i][2];
                if(map.size()==k){//缓存已满
                    //删除最久未使用
                    int delKey = tail.pre.k;
                    delete(tail.pre);
                    map.remove(delKey);
                }
                //加入新节点
                ListNode node = new ListNode(key, val);
                map.put(key, node);
                //更新缓存
                update(node);
            }else{//get(key)查询元素
                int key = operators[i][1];
                if(map.containsKey(key)){//查询得到
                    list.add(map.get(key).v);
                    //更新缓存
                    update(map.get(key));
                }else{//查询不到
                    list.add(-1);
                }
            }
        }
        
        //转为指定格式返回
        int[] res = new int[list.size()];
        for(int i=0; i<list.size(); i++){
            res[i] = list.get(i);
        }
        return res;
    }
    
    private void update(ListNode node){
        delete(node);//先删除双向链表节点
        node.next=head.next;//然后将改节点放到头部
        head.next.pre = node;
        head.next = node;
        node.pre = head;
    }
    
    private void delete(ListNode node){
        if(node.pre!=null) node.pre.next = node.next;
        if(node.next!=null) node.next.pre = node.pre;
    }
}

class ListNode{
    int k;
    int v;
    ListNode next, pre;
    public ListNode(int k, int v){
        this.k=k;
        this.v=v;
    }
}

NC108 最大正方形

题目地址：https://www.nowcoder.com/practice/0058c4092cec44c2975e38223f10470e?tpId=117&&tqId=37832&&companyId=239&rp=1&ru=/company/home/code/239&qru=/ta/job-code-high/question-ranking

方法一：动态规划

根据DP解题三步骤

  • 确定dp数组含义，此题dp[i][j]代表以坐标(i,j)的元素为右下角的正方形边长
  • 状态转移方程，dp[i][j]取决于dp[i-1][j]，dp[i-1][j-1]，dp[i][j-1]的最小值。即左正方形边长，左上正方形边长，上正方形边长三者的最小值
  • 边界，由于状态转移方程设计i-1,j-1，所以首行首列dp值要先确定

注意：最大边长初始值设为0，**在初始化首行首列dp值的时候如果存在字符1，应该更新最大边长为1**，否则首行首列有1，而剩余元素全为0的时候会错误。

代码

import java.util.*;

//和最大黑方阵不同，这题要求正方形全为1
public class Solution {
    public int solve (char[][] matrix) {
        //特例
        if(matrix.length==0 || matrix[0].length==0) return 0;
        
        int rows = matrix.length;
        int cols = matrix[0].length;
        int[][] dp = new int[rows][cols];
        int maxSquareLength = 0;
        //首行边界
        for(int i=0; i<rows; i++){
            if(matrix[i][0]=='1'){
                maxSquareLength=1;
                dp[i][0] = 1;
            }
        }
        //首列边界
        for(int i=0; i<cols; i++){
            if(matrix[0][i]=='1'){
                dp[0][i] = 1;
                maxSquareLength = 1;
            }
         }
        //其余位置
        for(int i=1; i<rows; i++){
            for(int j=1; j<cols; j++){
                if(matrix[i][j]=='1'){
                    //状态转移
                    dp[i][j] = Math.min(Math.min(dp[i-1][j-1],dp[i-1][j]),dp[i][j-1]) + 1;
                    //更新最大边长
                    if(dp[i][j] > maxSquareLength) maxSquareLength = dp[i][j];
                }
            }
        }
        //返回面积
        return maxSquareLength*maxSquareLength;
    }
}

import java.util.*;

public class Solution {
    public ArrayList<String> Permutation(String str) {
        char[] arr = str.toCharArray();
        
        //set集合实现排序和去重
        Set<String> set = new TreeSet<>();
        boolean[] isVisited = new boolean[arr.length];
        dfs(arr, isVisited, new StringBuilder(), set);
        //返回指定格式
        return new ArrayList<>(set);
    }
    
    private void dfs(char[] arr, boolean[] isVisited, StringBuilder sb, Set<String> set){
        //终止条件
        if(sb.length()==arr.length){
            set.add(sb.toString()); return;
        }
        
        //遍历字符数组+选择回溯
        for(int i=0; i<arr.length; i++){
            if(isVisited[i]) continue;
            
            sb.append(arr[i]);
            isVisited[i] = true;
            dfs(arr, isVisited, sb, set);
            sb.deleteCharAt(sb.length()-1);
            isVisited[i] = false;
        }
    }
}

NC127 最长公共子串

题目地址：https://www.nowcoder.com/practice/f33f5adc55f444baa0e0ca87ad8a6aac?tpId=117&&tqId=37799&&companyId=239&rp=1&ru=/company/home/code/239&qru=/ta/job-code-high/question-ranking

方法一：动态规划

设置动态规划数组dp[i][j]，表示以str1第i个字符，str2第j个字符为结尾的最长公共子串

转移方程

  • 首行首列，字符相等则dp[i][j]=1，否则为0
  • 其他位置，字符相等则为左上角结尾的最长公共子串长度加一，即dp[i][j]=dp[i-1][j-1]+1

在状态转移时，同时记录最长公共子串的长度和下标，最后根据结尾下标和长度返回最长公共子串

代码

import java.util.*;

public class Solution{
    public String LCS(String str1, String str2){
        int m=str1.length(), n=str2.length();
        if(str1==null || m==0 || str2==null || n==0) return "0";
        
        //记录最长公共子串末尾所有位置，用str1的索引
        int index=0, maxLen=0;
        //dp代表以(i,j)为结尾的LCS
        int[][] dp = new int[m][n];
        for(int i=0; i<m; i++){
            for(int j=0; j<n; j++){
                if(str1.charAt(i)==str2.charAt(j)){
                    if(i==0 || j==0){//首行，首列
                        dp[i][j] = 1;
                    }else{//其他
                        dp[i][j] = dp[i-1][j-1] + 1;
                    }
                    //更新最长公共子串和结尾下标
                    if(maxLen<dp[i][j]){
                        maxLen = dp[i][j];
                        index = i;
                    }
                }
            }
        }
        //结果
        if(maxLen==0) return "0";
        return str1.substring(index+1-maxLen, index+1);
    }
}

NC128 接雨水问题

题目地址：https://www.nowcoder.com/practice/31c1aed01b394f0b8b7734de0324e00f?tpId=117&&tqId=37802&&companyId=239&rp=1&ru=/company/home/code/239&qru=/ta/job-code-high/question-ranking

方法一：动态规划

对于下标i，下雨后能达到的最大高度等于下标i两步的最大高度的最小值，能接的雨水量等于能达到的最大高度减去height[i]

我们需要维护两个高度数组

  • leftMax，下标i到最左端的条形块的最大高度
  • rightMax，下标i到最右端的条形块的最大高度

然后扫描数组height并更新答案

  • 累加min(leftMax,rightMax)-height[i]

注意：结果用long保存，防止溢出！

代码

import java.util.*;

public class Solution {
      public long maxWater(int[] arr) {
          int len = arr.length;
          if(len<=2) return 0;
          
          //维护两个左右高度数组
          int[] leftMax = new int[len];
          int[] rightMax = new int[len];
          leftMax[0] = arr[0];
          for(int i=1; i<len; i++){
              leftMax[i] = Math.max(leftMax[i-1], arr[i]);
          }
          rightMax[len-1] = arr[len-1];
          for(int i=len-2; i>=0; i--){
              rightMax[i] = Math.max(rightMax[i+1], arr[i]);
          }
          
          //遍历得到每个位置接到的雨水
          long res = 0;
          for(int i=1; i<len-1; i++){
              res += Math.min(leftMax[i], rightMax[i])-arr[i];
          }
          return res;
      }
}

NC133 链表的奇偶重排

题目地址：https://www.nowcoder.com/practice/02bf49ea45cd486daa031614f9bd6fc3?tpId=117&&tqId=37845&&companyId=239&rp=1&ru=/company/home/code/239&qru=/ta/job-code-high/question-ranking

方法一：双指针

设置两个奇偶头指针oddHead和evenHead，设置odd指向奇数起点，设置even指向偶数起点

然后遍历链表，修改指针奇偶重排，最后将odd节点的next指向evenHead，返回奇数节点头指针oddHead

代码

import java.util.*;

public class Solution{
    public ListNode oddEvenList (ListNode head){
        if(head == null || head.next == null) return head;
        //odd 指向奇数节点的指针 oddHead 指向初始奇数节点的头指针
        //even 指向偶数节点的指针 evenHead 指向初始偶数节点的头指针
        ListNode odd = head, oddHead = head, even = head.next, evenHead = head.next;
        while (even!= null && even.next!=null){
            // 奇数节点指向偶数节点的 next
            odd.next = even.next;
            // 奇数节点指针后移
            odd = odd.next;
            // 偶数节点指向奇数节点的 next
            even.next = odd.next;
            // 偶数节点指针后移
            even = even.next;
        }
        // 将奇数节点的 next 指向 偶数节点的初始头指针
        odd.next = evenHead;
        // 返回奇数节点的初始头指针
        return oddHead;
    }
}

NC137 表达式求值

题目地址：https://www.nowcoder.com/practice/c215ba61c8b1443b996351df929dc4d4?tpId=117&companyId=239&rp=1&ru=%2Fcompany%2Fhome%2Fcode%2F239&qru=%2Fta%2Fjob-code-high%2Fquestion-ranking

方法一：栈

LeetCode227宫水三叶的解法「究极表达式计算」，对于这类问题，都可以套用这个思路进行解决。

对于任何「任何表达式」而言，我们都是用两个栈nums（数栈）和ops（操作符栈）

然后从前往后做，对遍历到的字符做分类讨论

  • 空格：跳过
  • (：直接加入ops中，等待与之匹配的)
  • )：使用现有的nums和ops进行计算，直到遇到最近的一个左括号为止，计算结果放到nums
  • 数字：从当前位置晚会读取，将一个连续数字整体取出，加入nums
  • +-*/^%：需要将操作符放入ops中。在放入之前先把栈内可以算的都算掉（当前运算符比栈顶运算符优先级低/同等，才进行运算），使用现有的nums和ops进行计算，直到没有操作符或遇到左括号，计算结果放到nums

一些细节

  • 由于第一个数可能是负数，为了减少边界判断，一个小技巧是先往nums添加一个0
  • 为防止()内出现的首个字符为运算符，将所有空格去掉，并将(-替换为(0-，(+替换为(0+，当然也可以不预处理而在循环中处理
  • 从理论上分析，nums最好存放的是long，而不是int，防止表达式中间结果或最终结果溢出

代码

import java.util.*;

class Solution {
    //使用Map维护一个运算符优先级
    Map<Character, Integer> map = new HashMap<Character, Integer>(){{
        put('-', 1);
        put('+', 1);
        put('*', 2);
        put('/', 2);
        put('%', 2);
        put('^', 3);
    }};

    public int solve(String s) {
        //表达式预处理
        s = s.replaceAll(" ", "");//去空格
        char[] arr = s.toCharArray();
        int n = arr.length;
        //数栈和操作符栈
        Deque<Integer> nums = new LinkedList<>();
        nums.offerLast(0);//先往nums加个0，防止负数开头
        Deque<Character> ops = new LinkedList<>();
        //循环处理
        for(int i=0; i<n; i++){
            char c = arr[i];
            if(c=='('){//左括号
                ops.offerLast(c);
            }else if(c==')'){//右括号
                //计算到最近的左括号
                while(!ops.isEmpty()){
                    if(ops.peekLast()!='('){
                        calc(nums, ops);
                    }else{
                        ops.pollLast(); break;
                    }
                }
            }else{
                if(Character.isDigit(c)){//数
                    int num=0, index=i;
                    //取出从i开始的数
                    while(index<n && Character.isDigit(arr[index])) num = num*10 + (arr[index++]-'0');
                    nums.offerLast(num);
                    i = index-1;
                }else{//操作符
                    if(i>0 && (arr[i-1]=='(')) nums.offerLast(0);//处理(-,(+

                    //操作符入栈之前，把能运算的先运算了
                    //栈顶运算符比当前运算符优先级高，进行运算
                    while(!ops.isEmpty() && ops.peekLast()!='('){
                        char pre = ops.peekLast();
                        if(map.get(pre)>=map.get(c)){
                            calc(nums, ops);
                        }else break;
                    }
                    ops.offerLast(c);
                }
            }
        }
        //将剩余的算完
        while(!ops.isEmpty()) calc(nums, ops);
        return nums.peekLast();
    }

    private void calc(Deque<Integer> nums, Deque<Character> ops){
        if(nums.isEmpty() || nums.size()<2) return;
        if(ops.isEmpty()) return;
        int b = nums.pollLast(), a = nums.pollLast();
        char op = ops.pollLast();
        int ans = 0;
        if(op=='+') ans = a+b;
        else if(op=='-') ans = a-b;
        else if(op=='*') ans = a*b;
        else if(op=='/') ans = a/b;
        else if(op=='^') ans = (int)Math.pow(a,b);
        else if(op=='%') ans = a%b;
        nums.offerLast(ans);
    }
}

NC149 KMP算法

题目地址：https://www.nowcoder.com/practice/bb1615c381cc4237919d1aa448083bcc?tpId=117&&tqId=37859&&companyId=239&rp=1&ru=/company/home/code/239&qru=/ta/job-code-high/question-ranking

方法一：KMP匹配算法

阮一峰博客介绍了KMP算法中next数组的理解：http://www.ruanyifeng.com/blog/2013/05/Knuth%E2%80%93Morris%E2%80%93Pratt_algorithm.html

next数组（部分匹配值）的计算，它就是前缀和后缀的最长共有元素，如下图

  • A的前缀和后缀都为空集，最长共有元素长度为0
  • AB的前缀为[A]，后缀为[B]，最长共有元素长度为0
  • ABC的前缀为[A,AB]，后缀为[BC,B]，最长共有元素长度为0
  • ABCD的前缀为[A,AB,ABC]，后缀为[BCD,CD,D]，最长共有元素长度为0
  • ABCDA的前缀为[A,AB,ABC,ABCD]，后缀为[BCDA,CDA,DA,A]，最长共有元素长度为1
  • ABCDAB的前缀为[A,AB,ABC,ABCD,ABCDA]，后缀为[BCDAB,CDAB,DAB,AB,B]，最长共有元素长度为2
  • ABCDABD的前缀为[A,AB,ABC,ABCD,ABCDA,ABCDAB]，后缀为[BCDABD,CDABD,DABD,ABD,BD,B]，最长共有元素长度为0

本题的基本步骤

  • 首先是特殊情况判断，模式串S长度大于主串T或者主串T长度为0，直接返回0
  • 然后获取模式串的next数组
  • 最后分别遍历主串和模式串，只要当前字符相等，模式串和主串均后移一位，如果不相等，模式串重新回退到next数组指定的下一位索引。当模式串索引达到长度m时，匹配次数+1，同时将模式串回退到next数组指定的下一位索引

代码

import java.util.*;

public class Solution{
    public int kmp(String S, String T){
        //特殊情况判断
        int m=S.length(), n=T.length();
        if(m>n || n==0) return 0;
        
        //初始化计数，获取next数组
        int count=0;
        int[] next = getNext(S);
        //遍历主串和模式串
        for(int i=0,j=0; i<n; i++){
            //只要不相等，回退到next数组记录的下一位
            while(j>0 && T.charAt(i)!=S.charAt(j)) j = next[j-1];
            //前缀索引后移
            if(T.charAt(i)==S.charAt(j)) j++;
            //如果j为m，说明完全匹配一次
            if(j==m){
                //计数加一，索引回退到next数组记录的下一位
                count++;
                j = next[j-1];
            }
        }
        return count;
    }
    
    //获取next数组
    public int[] getNext(String S){
        int m = S.length();
        int[] next = new int[m];
        for(int i=1, j=0; i<m; i++){
            //只要不相等，回退到next数组记录的下一位
            while(j>0 && S.charAt(i)!=S.charAt(j)) j = next[j-1];
            //前缀索引后移
            if(S.charAt(i)==S.charAt(j)) j++;
            //确定应该回退到的下一个索引
            next[i] = j;
        }
        return next;
    }
}
