HJ6 质数因子

题目地址：https://www.nowcoder.com/practice/196534628ca6490ebce2e336b47b3607?tpId=37&&tqId=21229&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：迭代

分解质因数，从最小的开始（2）不停做除法，除到为1。或者除到最后除不了，就输出自己。还可以判断下除数是不是质数，但没必要，还会增加判断次数，得不偿失。

每找到一个质因数，就不断除以它，直到除尽，这样保证找到的因数一定是质数。

「有两个坑」

  • 这个数本身就是个质数，输出自己。
  • 可能超时，和判断质数一样，没必要从2一直尝试到number，循环到Math.sqrt(number)即可。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        long number = sc.nextLong();
        //从2开始，判断到sqrt(number)即可
        for(int i=2; i<=Math.sqrt(number); i++){
            //每找到一个质因数，除尽为止，这样保证找到的因数一定是质数
            while(number%i==0){
                System.out.print(i+" ");
                number /= i;
            }
        }
        //最后number不为1，说明除完后剩余一个质数
        System.out.println(number==1 ? "":number+" ");
    }
}

HJ16 购物单

题目地址：https://www.nowcoder.com/practice/f9c6f980eeec43ef85be20755ddbeaf4?tpId=37&&tqId=21239&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：动态规划

这篇博客很生动的讲述了背包问题的动态规划思想：https://www.jianshu.com/p/a66d5ce49df5，看完博客，再来看题目就很好做，购物单问题实际跟博客里的背包问题差不多。

在购物问题里，先处理主件行数据，需要考虑一下四种情况：

  • 主件
  • 主件+附件1
  • 主件+附件2
  • 主件+附件1+附件2

主件行数据从以上四种情况里挑选出最优的一种情况，然后它的附件行只需要照抄主件行数据就行。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        //输入数据
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();//总钱数
        int m = sc.nextInt();//商品数
        Goods[] goods = new Goods[m+1];
        for(int i=1; i<=m; i++){
            int v = sc.nextInt();//价格
            int p = sc.nextInt();//重要程度
            int q = sc.nextInt();//附件
            goods[i] = new Goods(v, p, q);
            //如果商品是q的附件，默认到商品i时，商品q已经放入数组（即先出现主件，再出现附件）
            if(q>0){
                if(goods[q].addID1==0) goods[q].addID1=i;//商品q还没添加附件
                else goods[q].addID2=i;//商品q已经添加一个附件
            }
        }
        
        //处理网格
        int[][] dp = new int[m+1][n+1];
        for(int i=1; i<=m; i++){//第i个商品
            int v1=0, v2=0, v3=0, v4=0;
            int w1=0, w2=0, w3=0, w4=0;
            
            //1.主件
            v1 = goods[i].v;
            w1 = v1*goods[i].p;
            
            //2.主件+附件1
            if(goods[i].addID1!=0){
                v2 = v1 + goods[goods[i].addID1].v;
                w2 = w1 + goods[goods[i].addID1].v*goods[goods[i].addID1].p;
            }
            
            //3.主件+附件2
            if(goods[i].addID2!=0){
                v3 = v1 + goods[goods[i].addID2].v;
                w3 = w1 + goods[goods[i].addID2].v*goods[goods[i].addID2].p;
            }
            
            //4.主件+附件1+附件2
            if(goods[i].addID1!=0 && goods[i].addID2!=0){
                v4 = v2+v3-v1;
                w4 = w2+w3-w1;
            }
            
            for(int j=1; j<=n; j++){
                if(goods[i].q>0){
                    //附件行数据会和它的主件行数据一模一样，直接拷贝上一行数据即可
                    dp[i][j] = dp[i-1][j];
                }else{
                    //对于主件行数据而言，它需要考虑四种情况里，自己选择哪种能够利益最大化
                    dp[i][j] = dp[i-1][j];
                    if(v1<=j) dp[i][j] = Math.max(dp[i][j], w1+dp[i-1][j-v1]);
                    if(v2<=j) dp[i][j] = Math.max(dp[i][j], w2+dp[i-1][j-v2]);
                    if(v3<=j) dp[i][j] = Math.max(dp[i][j], w3+dp[i-1][j-v3]);
                    if(v4<=j) dp[i][j] = Math.max(dp[i][j], w4+dp[i-1][j-v4]);
                }
            }
        }
        //输出数据
        System.out.println(dp[m][n]);
    }
}

//商品类
class Goods{
    int v;
    int p;
    int q;
    
    int addID1=0;//附件1的ID
    int addID2=0;//附件2的ID
    
    public Goods(int v, int p, int q){
        this.v=v;
        this.p=p;
        this.q=q;
    }
}

HJ18 识别有效的IP地址和掩码并进行分类统计

题目地址：https://www.nowcoder.com/practice/de538edd6f7e4bc3a5689723a7435682?tpId=37&&tqId=21241&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：模拟

注意事项

  • Java的split正则匹配加上前缀"\\"；
  • 掩码拼接成32位字符串，每个点分十进制位Integer.toBinaryString()不包含前导0，要拼接成8位

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int[] arr = new int[7];
        while(sc.hasNext()){
            String[] str = sc.nextLine().split("\\~");
            if(isValidIP(str[0]) && isValidMask(str[1])){
                //有效IP和掩码，分类统计
                int num = Integer.parseInt(str[0].split("\\.")[0]);
                if(num>=1 && num<=126) arr[0]++;
                if(num>=128 && num<=191) arr[1]++;
                if(num>=192 && num<=223) arr[2]++;
                if(num>=224 && num<=239) arr[3]++;
                if(num>=240 && num<=255) arr[4]++;
                
                int num2 = Integer.parseInt(str[0].split("\\.")[1]);
                if(num==10) arr[6]++;
                if(num==172 && (num2>=16&&num2<=31)) arr[6]++;
                if(num==192 && num2==168) arr[6]++;
            }else{//错误IP或掩码
                arr[5]++;
            }
        }
        //输出结果
        for(int i=0; i<7; i++){
            if(i!=6) System.out.print(arr[i]+" ");
            else System.out.print(arr[i]);
        }
    }
    
    //是否有效IP
    private static boolean isValidIP(String ip){
        String[] str = ip.split("\\.");
        //长度是否正确
        if(str.length!=4) return false;
        
        //数字范围是否正确
        int num=0;
        for(int i=0; i<4; i++){
            num = Integer.parseInt(str[i]);
            if(num<0 || num>255) return false;
        }
        return true;
    }
    
    //是否有效掩码
    private static boolean isValidMask(String mask){
        //掩码拼接成32位字符串
        String[] str = mask.split("\\.");
        StringBuilder sb = new StringBuilder();
        for(int i=0; i<4; i++){
            String temp = Integer.toBinaryString(Integer.parseInt(str[i]));
            for(int j=0; j<8-temp.length(); j++){
                temp = "0"+temp;
            }
            sb.append(temp);
        }
        //全1全0情况
        if(sb.indexOf("0")==-1 || sb.indexOf("1")==-1) return false;
        
        if(sb.indexOf("0")<sb.lastIndexOf("1")) return false;
        return true;
    }
}

HJ19 简单错误记录

题目地址：https://www.nowcoder.com/practice/2baa6aba39214d6ea91a2e03dff3fbeb?tpId=37&&tqId=21242&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：哈希表

用LinkedHashMap记录错误，保证迭代输出顺序，输出最后8条错误，错误不满8条则全部输出。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        //用集合记录错误
        Map<String, Integer> map = new LinkedHashMap<>();
        String file="";
        String key="";
        while(sc.hasNext()){
            String[] str = sc.nextLine().split("\\\\");
            String[] error = str[str.length-1].split("\\ ");
            if(error[0].length()>16) file = error[0].substring(error[0].length()-16);
            else file = error[0];
            key = file+" "+error[1];
            map.put(key, map.getOrDefault(key, 0)+1);
        }
        
        //输出最后8个错误，不满足8个则全部输出
        int len = map.size();
        Set<String> set = map.keySet();
        if(len>8){
            int i=len-8;
            for(String k : set){
                if(i>0) i--;
                else System.out.println(k+" "+map.get(k));
            }
        }else{
            for(String k : set){
                System.out.println(k+" "+map.get(k));
            }
        }
    }
}

HJ20 密码验证合格程序

题目地址：https://www.nowcoder.com/practice/184edec193864f0985ad2684fbc86841?tpId=37&&tqId=21243&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：字符串

关键是第三条，不能有相同长度大于2的子串重复。从0~keys.length()-4取3位，看取位置后一位开始的剩余子串是否包含3位重复（有3位重复就是有相同长度大于2的子串重复）。

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            String keys = sc.nextLine();
            if(keys.length()<=8){//长度<=8
                System.out.println("NG");
            }else{
                if(check2(keys) && check3(keys)){
                    System.out.println("OK");
                }else{
                    System.out.println("NG");
                }
            }
            
        }
    }
    
    //要求2判断
    private static boolean check2(String keys){
        int a=0, b=0, c=0, d=0;
        for(int i=0; i<keys.length(); i++){
            char ch = keys.charAt(i);
            if(ch>='A' && ch<='Z') a=1;
            else if(ch>='a' && ch<='z') b=1;
            else if(ch>='0' && ch<='9') c=1;
            else d=1;
        }
        if(a+b+c+d>=3) return true;
        else return false;
    }
    
    //关键是第三点的判断，巧妙的用了substring和contains
    private static boolean check3(String keys){
        for(int i=0; i<keys.length()-3; i++){
            if(keys.substring(i+1).contains(keys.substring(i, i+3))) return false;
        }
        return true;
    }
}

HJ23 删除字符串中出现次数最少的字符

题目地址：https://www.nowcoder.com/practice/05182d328eb848dda7fdd5e029a56da9?tpId=37&&tqId=21246&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：迭代

分三步即可

  • 统计字符出现次数；
  • 找到最小出现次数min；
  • 遍历字符串删除等于min的最小字符串，StringBuilder拼接结果。

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            String str = sc.nextLine();
            //统计字符次数
            int[] arr = new int[26];
            for(int i=0; i<str.length(); i++){
                arr[str.charAt(i)-'a']++;
            }
            //最小出现次数是多少
            int min=str.length();
            for(int i=0; i<26; i++){
                if(arr[i]!=0 && min>arr[i]) min = arr[i];
            }
            //删除+拼接
            StringBuilder sb = new StringBuilder();
            for(int i=0; i<str.length(); i++){
                char ch = str.charAt(i);
                if(arr[ch-'a']!=min) sb.append(ch);
            }
            //输出结果
            System.out.println(sb.toString());
        }
    }
}

HJ24 合唱队

题目地址：https://www.nowcoder.com/practice/6d9d69e3898f45169a441632b325c7b4?tpId=37&&tqId=21247&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：动态规划

最长子序列问题

  • **先找到每个位置i左侧的最长上升子序列长度numL[i]**：等于左侧比它小的所有位置的最长子序列的最大值+1
  • **再找到每个位置i右侧的最长下降子序列长度numR[i]**：等于右侧比它小的所有位置的最长子序列的最大值+1
  • 然后求出所有位置的最长序列长度：等于左侧最长子序列长度+右侧最长子序列长度-1（因为该位置被算了两次，所以减1）
  • 最后用数组长度减去最长序列长度就是需要出队的同学数

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] height = new int[n];
            for(int i=0; i<n; i++){
                height[i] = sc.nextInt();
            }
            //每个位置左侧最长上升子序列
            int[] numL = new int[n];
            for(int i=0; i<n; i++){
                for(int j=0; j<i; j++){//左侧比height[i]小的位置最长上升序列长度
                    if(height[i]>height[j]) numL[i] = Math.max(numL[i], numL[j]);
                }
                numL[i] += 1;//左侧比height[i]小位置最长上升序列长度+1
            }
            //每个位置右侧最长下降子序列
            int[] numR = new int[n];
            for(int i=n-1; i>=0; i--){
                for(int j=n-1; j>i; j--){//右侧比heigth[i]小的位置最长下降序列长度
                    if(height[i]>height[j]) numR[i] = Math.max(numR[i], numR[j]);
                }
                numR[i] += 1;//右侧比height[i]小位置最长下降序列长度+1
            }
            //根据每个位置最长合唱队numL[i]+numR[i]-1，求出所有位置中最大值
            int maxLen=0;
            for(int i=0; i<n; i++){
                if(numL[i]+numR[i]-1>maxLen) maxLen = numL[i]+numR[i]-1;
            }
            //最后n-maxLen即为需要出列人数
            System.out.println(n-maxLen);
        }
    }
}

HJ25 数据分类处理

题目地址：https://www.nowcoder.com/practice/9a763ed59c7243bd8ab706b2da52b7fd?tpId=37&&tqId=21248&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：集合

理解题意需要花费一段时间，题目并不难。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int m = sc.nextInt();
            int[] arrI = new int[m];
            for(int i=0; i<m; i++){
                arrI[i] = sc.nextInt();
            }
            int n = sc.nextInt();
            int[] arrR = new int[n];
            for(int i=0; i<n; i++){
                arrR[i] = sc.nextInt();
            }
            //R序列排序
            Arrays.sort(arrR);
            //遍历规则R，找到符合条件保存
            List<Integer> res = new ArrayList<>();
            for(int i=0; i<n; i++){
                if(i>0 && arrR[i]==arrR[i-1]) continue;//跳过重复规则
                
                fun(arrI, arrR[i], res);
            }
            //输出结果
            res.add(0, res.size());
            for(int i=0; i<res.size(); i++){
                if(i!=res.size()-1) System.out.print(res.get(i)+" ");
                else System.out.println(res.get(i));
            }
        }
    }
    
    private static void fun(int[] arrI, int x, List<Integer> res){
        //遍历找到符合规则的保存下标
        String rule = String.valueOf(x);
        List<Integer> temp = new ArrayList<>();
        for(int i=0; i<arrI.length; i++){
            if(String.valueOf(arrI[i]).contains(rule)){
                temp.add(i);
            }
        }
        //按要求移动到最终结果集合res中
        if(temp.size()>0){
            res.add(x);//加入规则
            res.add(temp.size());//满足规则的个数
            for(int i=0; i<temp.size(); i++){
                res.add(temp.get(i));//加入下标
                res.add(arrI[temp.get(i)]);//加入值
            }
        }
    }
}

HJ26 字符串排序

题目地址：https://www.nowcoder.com/practice/5190a1db6f4f4ddb92fd9c365c944584?tpId=37&&tqId=21249&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：稳定排序

这道题的关键是前两条规则，合并成一句话就是字母以ASCII码从小到大排序，并且不区分大小写，同时要求按照输入顺序排列（即排序是稳定的！）解决方案有两种

  • 遍历字符串，StringBuilder拼接字符串中字母，转为charArray做稳定排序（如冒泡，判断条件改为同时转为大写或小写判断大写），最后遍历拼接最终字符串。
  • 遍历字符串，将字母放入list集合，利用Collections.sort方法+自定义比较器完成排序，最后遍历拼接最终字符串。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            String str = sc.nextLine();
            String res = sortStr(str);
            System.out.println(res);
        }
    }
    
    private static String sortStr(String str){
        //将所有字母加入集合
        List<Character> list = new ArrayList<>();
        for(int i=0; i<str.length(); i++){
            char c = str.charAt(i);
            if((c>='a'&&c<='z') || (c>='A'&&c<='Z')){
                list.add(c);
            }
        }
        //按照规则排序，也可以自己写一个稳定的排序算法！
        Collections.sort(list, new Comparator<Character>(){
            public int compare(Character a, Character b){
                return Character.toLowerCase(a)-Character.toLowerCase(b);
//                 if((a>'Z'&&b>'Z') || (a<='Z'&&b<='Z')){//ab小写 或 ab大写
//                     return a-b;
//                 }else{//a,b一大一小
//                     if(a>'Z'){
//                         return 'A'+a-'a'-b;
//                     }
//                     return a-('A'+b-'a');
//                 }
            }
        });
        //拼接结果
        StringBuilder sb = new StringBuilder();
        int index=0;
        for(int i=0; i<str.length(); i++){
            char c = str.charAt(i);
            if((c>='a'&&c<='z') || (c>='A'&&c<='Z')){
                sb.append(list.get(index));
                index++;
            }else{
                sb.append(c);
            }
        }
        return sb.toString();
    }
}

HJ28 素数伴侣

题目地址：https://www.nowcoder.com/practice/b9eae162e02f4f928eac37d7699b352e?tpId=37&&tqId=21251&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：匈牙利算法

素数除了2，其他是奇数，题目数组取值范围[2,30000]，两数的和不可能为2，所以只考虑奇数素数。要得到奇数，只能是奇数+偶数，这就得到一个二分图，对二分图做最大匹配，得到最多的匹配对数，有一个经典的算法，匈牙利算法。

二分图：你可以把图上点分成两堆，每堆之间不会有边，两个堆之间才可能连边。本题分成奇数和偶数两堆，刚好是个二分图问题。

代码

import java.util.*;

public class Main{
    //[2,30000]范围，素数不是偶数，和是素数只能是奇数+偶数，二分图最大匹配，匈牙利算法
    //匈牙利算法，先到先得 能让就让。有机会上，没机会创造机会也要上
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] arr = new int[n];
            for(int i = 0; i < n; i++){
                arr[i] = sc.nextInt();
            }
            //分两组，奇/偶
            ArrayList<Integer> evens = new ArrayList<Integer>();
            ArrayList<Integer> odds = new ArrayList<Integer>();
            for(int i = 0; i < n; i++){
                if((arr[i] & 1) != 1){
                    evens.add(arr[i]);
                }else{
                    odds.add(arr[i]);
                }
            }
            //下面开始才是重头戏
            //当前下标偶数的奇数匹配
            int[] evensMatch = new int[evens.size()];
            int result = 0;
            //遍历奇数去匹配偶数
            for(int i = 0; i < odds.size(); i++){
                //每一步重新创建，也就是相当于清空，boolean数组用于标记某个某数位置是否已匹配
                boolean[] isVisited = new boolean[evens.size()];
                //这里采用了匈牙利算法，先到先得
                if(find(odds.get(i), evens, isVisited, evensMatch)){
                    result++;
                }
            }
            System.out.println(result);
        }
    }
    
    public static boolean isPrime(int num){
        for(int i = 2; i * i <= num; i++){
            if(num % i == 0) {
                return false;
            }
        }
        return true;
    }
    
    public static boolean find(int x, ArrayList<Integer> evens, boolean[] isVisited, int[] evensMatch){
        //遍历偶数，去检查当前传入的奇数能否与偶数哪些数匹配
        for(int i = 0; i < evens.size(); i++){
            //如果当前偶数与传入的奇数匹配，并且当前偶数位还没有匹配过奇数
            if(isPrime(x + evens.get(i)) && !isVisited[i]){
                //设置当前偶数位匹配为true，也就是 1
                isVisited[i] = true;
                //如果第i个偶数没有伴侣，或者第i个偶数原来有伴侣，并且该伴侣能够重新找到伴侣的话(这里有递归调用)
                //则奇数x可以设置为第i个偶数的伴侣
                //这里采用了匈牙利算法，能让则让 
                if(evensMatch[i] == 0 || find(evensMatch[i], evens, isVisited, evensMatch)) {
                    evensMatch[i] = x;
                    return true;
                }
            }
        }
        //遍历完偶数都没有可以与传入奇数做伴侣的，该奇数只能孤独终老了
        return false;
    }

HJ41 称砝码

题目地址：https://www.nowcoder.com/practice/f9a4c19050fc477e9e27eb75f3bfd49c?tpId=37&&tqId=21264&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：哈希集合

将所有砝码放入集合。设置一个去重集合HashSet，先放入0，然后遍历砝码集合，每遍历一个砝码就拿HashSet中现有元素与该砝码值相加，得到新增加的重量集合，然后把新增加的重量再放入HashSet集合中，遍历结束HashSet的大小就是可以称的重量数目。

例如有2个重量1g和2g的砝码

  • 当遍历第一个砝码{0}->{0, 0+1}->{0, 1}

  • 当遍历第二个砝码{0, 1}->{0, 1, 0+2, 1+2}->{0, 1, 2, 3}

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] weights = new int[n];
            int[] counts = new int[n];
            for(int i=0; i<n; i++){
                weights[i] = sc.nextInt();
            }
            for(int i=0; i<n; i++){
                counts[i] = sc.nextInt();
            }
            //将所有砝码放入集合中
            List<Integer> list = new ArrayList<>();
            for(int i=0; i<n; i++){
                for(int j=0; j<counts[i]; j++){
                    list.add(weights[i]);
                }
            }
            //计算不同重量数
            int sum = countWeights(list);
            System.out.println(sum);
        }
    }
    
    private static int countWeights(List<Integer> list){
        //重量集合，先放入0
        Set<Integer> weights = new HashSet<>();
        weights.add(0);
        //遍历砝码，每遍历一个就与现有重量集合所有元素相加，得到新增加的重量
        for(int i=0; i<list.size(); i++){
            Set<Integer> newWeigths = new HashSet<>();
            for(int j : weights){//得到新增加的重量集合
                newWeigths.add(j + list.get(i));
            }
            for(int j : newWeigths){//新增加的重量放入总集合中
                weights.add(j);
            }
        }
        return weights.size();
    }
}

方法一：深度优先搜索

典型的dfs，设置好结束条件和剪枝条件，选择和撤销选择。注意点是，选择后将网格值修改为障碍1，撤销选择后修改为可走0，能够节省一个isVisited布尔数组。

//回溯
import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            int n = sc.nextInt();
            int m = sc.nextInt();
            int[][] maze = new int[n][m];
            for(int i=0; i<n; i++){
                for(int j=0; j<m; j++){
                    maze[i][j] = sc.nextInt();
                }
            }
            //深度优先搜索
            List<List<int[]>> ans = new ArrayList<>();
            List<int[]> path = new ArrayList<>();
            dfs(maze, 0, 0, path, ans);
            //题目保证了唯一，这里还是找一下最短路径
            int minPath = Integer.MAX_VALUE;
            int minIndex = 0;
            for(int i=0; i<ans.size(); i++){
                int size = ans.get(i).size();
                if(minPath>size){
                    minIndex=i;
                    minPath = size;
                }
            }
            //输出最短路径
            for(int[] arr : ans.get(minIndex)){
                System.out.println("("+arr[0]+","+arr[1]+")");
            }
        }
    }
    
    private static void dfs(int[][] maze, int i, int j, List<int[]> path, List<List<int[]>> ans){
        //结束条件，一定要用路径新建集合，path最后会remove为空
        if(i==maze.length-1 && j==maze[0].length-1){
            path.add(new int[]{i, j});
            ans.add(new ArrayList<>(path));
            return;
        }
        //边界条件，剪枝
        if(i<0 || i>=maze.length || j<0 || j>=maze[0].length || maze[i][j]==1) return;
        
        //i,j加入集合,并且当前位置设置为障碍
        path.add(new int[]{i, j});
        maze[i][j]=1;
        //向四个方向递归
        dfs(maze, i+1, j, path, ans);
        dfs(maze, i, j+1, path, ans);
        dfs(maze, i-1, j, path, ans);
        dfs(maze, i, j-1, path, ans);
        //找到死胡同，回溯，撤销i,j选择，当前位置设置为可走
        path.remove(path.size()-1);
        maze[i][j]=0;
    }
}

HJ44 Sudoku

题目地址：https://www.nowcoder.com/practice/78a1a4ebe8a34c93aac006c44f6bf8a1?tpId=37&&tqId=21267&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：深度优先搜索

大致步骤如下

  • 从(0,0)开始搜索dfs(arr, 0, 0)，策略是从左到右，从上到下

    • 如果搜索行等于9，结束，返回true
    • 如果搜索列等于9，返回dfs(arr, row+1, 0)
    • 如果当前位置不为0，搜索下一列，dfs(arr, row, col+1)

  • 如果不满足上述条件，则是值为0的格子，从1-9尝试填入，判断填入是否满足同行同列同九宫格条件，同时递归其它0格子填充时不冲突

    • 同时满足，返回true，否则，撤销选择arr[row][col]=0，尝试其他填充

  • 如果1-9填充都不行，返回false

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //初始化数独
            int[][] arr = new int[9][9];
            for(int i=0; i<9; i++){
                for(int j=0; j<9; j++){
                    arr[i][j] = sc.nextInt();
                }
            }
            //调用处理函数
            dfs(arr, 0, 0);
            //输出结果
            for(int i=0; i<9; i++){
                for(int j=0; j<9; j++){
                    System.out.print(arr[i][j]);
                    if(j!=8){
                        System.out.print(" ");
                    }
                }
                System.out.println();
            }
        }
    }
    
    //dfs
    private static boolean dfs(int[][] arr, int row, int col){
        //终止和剪枝
        if(row==9) return true;
        if(col==9) return dfs(arr, row+1, 0);
        if(arr[row][col]!=0) return dfs(arr, row, col+1);
        
        //尝试填充1-9
        for(int i=1; i<=9; i++){
            //选择i填充
            arr[row][col] = i;
            //是否满足同行同列通九宫格条件、递归处理其他位置
            if(check(arr, row, col) && dfs(arr, row, col+1)){
                return true;
            }
            //撤销选择，回溯
            arr[row][col] = 0;
        }
        return false;
    }
    
    //判断填入数字的同行同列同九宫格是否符合条件
    private static boolean check(int[][] arr, int row, int col){
        //同行
        for(int i=0; i<9; i++){
            if(i!=col && arr[row][i]==arr[row][col]) return false;
        }
        
        //同列
        for(int i=0; i<9; i++){
            if(i!=row && arr[i][col]==arr[row][col]) return false;
        }
        
        //同九宫格
        int up = row/3*3;
        int left = col/3*3;
        for(int i=up; i<up+3; i++){
            for(int j=left; j<left+3; j++){
                if(i!=row && j!=col && arr[i][j]==arr[row][col]) return false;
            }
        }
        
        return true;
    }
}

        newNode.next = head.next;
        head.next = newNode;
    }

        Node cur = head;
        while(cur.next!=null && cur.next.val!=val){
            cur = cur.next;
        }
        if(cur.next!=null) cur.next = cur.next.next;
        return head;
    }
}

HJ52 计算字符串的距离

题目地址：https://www.nowcoder.com/practice/3959837097c7413a961a135d7104c314?tpId=37&&tqId=21275&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：动态规划

看到字符串修改代价第一时间想到动态规划

传递公式

  • dp[i][j]表示字符串a的[1...i]和字符串b的[1...j]的字符串距离
  • 插入和删除互为逆过程，a删除指定字符变b，等同于b插入指定字符
  • 如果a[i]==b[j]，则说明a[i]和b[j]分别加入a，b之后不会影响字符串距离，dp[i][j]=dp[i-1][j-1]
  • 如果a[i]!=b[j]，则需要考虑三种可能，取最小值
    • a中插入字符，即dp[i][j]=dp[i-1][j]+1
    • b中插入字符，即dp[i][j]=dp[i][j-1]+1
    • a[i]替换成b[j]，即dp[i][j]=dp[i-1][j-1]+1

初始条件

  • 第一行，a为空，初始值等于列数
  • 第一列，b为空，初始值等于行数

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //调用距离计算函数
            int res = getDistance(sc.next(), sc.next());
            System.out.println(res);
        }
    }
    
    private static int getDistance(String str1, String str2){
        //特例
        if(str1.length()==0) return str2.length();
        if(str2.length()==0) return str1.length();
        
        //动态规划矩阵
        int len1=str1.length(), len2=str2.length();
        int[][] dp = new int[len1+1][len2+1];
        //初始值，首行首列
        for(int i=0; i<=len2; i++){
            dp[0][i] = i;
        }
        for(int i=0; i<=len1; i++){
            dp[i][0] = i;
        }
        //逐行推出dp[i][j]
        for(int i=1; i<=len1; i++){
            for(int j=1; j<=len2; j++){
                if(str1.charAt(i-1)==str2.charAt(j-1)){//相等，拷贝上一行前一列
                    dp[i][j]=dp[i-1][j-1];
                }else{//不相等，分三种情况，a中插入字符，b中插入字符，a[i]替换成b[j]
                    dp[i][j]=Math.min(Math.min(dp[i-1][j], dp[i][j-1]), dp[i-1][j-1]) + 1;
                }
            }
        }
        //返回字符串距离
        return dp[len1][len2];
    }
}

方法一：递归

令(m,n)表示将m个苹果放入n个盘子中的摆放方法总数

  • 当n>m时，必定有n-m个盘子永远空着，去掉它们对结果无影响，只需求(m,m)

  • 当n<=m时，放苹果分为两种情况，一种是有盘子为空，一种是每个盘子上都有苹果。

    • 假设有一个盘子为空，则(m,n)问题转化为m个苹果放到n-1个盘子上，即求(m,n-1)即可
    • 假设所有盘子都装有苹果，则盘子上至少有一个苹果，即最多剩下m-n个苹果，问题转化为m-n个苹果放到n个盘子上，即求(m-n, n)

难点理解，为什么是一个盘子而不是至少一个呢？因为至少一个包含在内，递归实现！

终止条件，递归两条路一条m会减少，一条n会减少，当苹果为0或盘子为1，返回1。

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            int counts = getCounts(sc.nextInt(), sc.nextInt());
            System.out.println(counts);
        }
    }
    
    private static int getCounts(int m, int n){
        //终止条件，苹果为0或盘子为1，只有一种放法
        if(m==0 || n==1) return 1;
        
        //盘子数大于苹果树，始终有n-m个盘子为空，返回(m, m)
        if(m<n) return getCounts(m, m);
        else{//m>=n，2种情况，至少一个盘子为空，盘子都有苹果
            return getCounts(m, n-1)+getCounts(m-n, n);
        }
    }

方法一：深度优先搜索

类似全排列的深度优先搜索，使用任意一个数字作为起始值，和一个标识数字是否已访问布尔数组，遍历数字进行递归和回溯即可

终止条件，数字使用完且计算结果为24，返回true

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        while(in.hasNext()){
            //处理输入
            String[] a = in.nextLine().split(" ");
            int[] nums = new int[4];
            for(int i = 0; i < 4; i ++){
                nums[i] = Integer.parseInt(a[i]);
            }
            //isVisited标识是否访问过
            boolean[] isVisited = new boolean[4];
            //以任意一个数作为开始
            boolean res = false;
            for(int i=0; i<4; i++){
                isVisited[i]=true;
                if(dfs(nums, 1, nums[i], isVisited)){
                    res=true;
                    break;
                }
                //以i开始无法得到24，撤销访问，尝试下一个数字开始
                isVisited[i]=false;
            }
            //输出结果
            System.out.println(res);
        }
    }

    private static boolean dfs(int[] nums, int counts, double val, boolean[] isVisited){
        //终止条件
        if(val == 24 && counts == 4) return true;
        
        //递归
        for(int i = 0; i < 4; i ++){
            if(!isVisited[i]){//未使用数字
                //选择数字使用
                isVisited[i] = true;
                if(dfs(nums, counts+1, val+nums[i], isVisited) || //加
                   dfs(nums, counts+1, val-nums[i], isVisited) || //减
                   dfs(nums, counts+1, val*nums[i], isVisited) || //乘
                   dfs(nums, counts+1, val/nums[i], isVisited))   //除
                      return  true;
                //撤销数字选择
                isVisited[i] = false;
            }
        }
        return false;
    }
}

HJ71 字符串通配符

题目地址：https://www.nowcoder.com/practice/43072d50a6eb44d2a6c816a283b02036?tpId=37&&tqId=21294&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：递归

采用递归思路，从前向后依次匹配

  • 遇到相同字符，不区分大小写，都向后移动一个字符
  • **遇到通配符?**，匹配一个字符（字母，数字或.），都向后移动一个字符
  • **遇到通配符***，三种情况，匹配（0个，1个，任意多个字符）
    • 匹配0个，通配符向后移动一个字符，字符串不动
    • 匹配1个，都向后移动一个字符
    • 匹配多个，通配符不动，字符串向后移动一个字符

递归的终止条件

  • 如果通配符和字符串同时到达末尾，返回true
  • 如果通配符或字符串其中一个到达末尾（不同时），返回false

注意事项

  • **连续的***，跳过之前的，只处理最后一个，否则会超时！
  • *和?只能匹配字母，数字或.

代码

import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            boolean res = dfs(sc.nextLine(), sc.nextLine(), 0, 0);
            System.out.println(res);
        }
    }
    
    private static boolean dfs(String s1, String s2, int p1, int p2){
        //终止条件
        if(p1==s1.length() && p2==s2.length()) return true;
        else if(p1==s1.length() || p2==s2.length()) return false;
        
        //遇到字符相等，不区分大小写
        if(Character.toLowerCase(s1.charAt(p1))==Character.toLowerCase(s2.charAt(p2))){
            //均后移
            return dfs(s1, s2, p1+1, p2+1);
        }else if(s1.charAt(p1)=='?'){//遇到字符?
            //如果匹配的不是字符数字或.，返回false
            if(!(Character.isLetterOrDigit(s2.charAt(p2)) || s2.charAt(p2)=='.')) return false;
            //否则，均后移一位
            return dfs(s1, s2, p1+1, p2+1);
        }else if(s1.charAt(p1)=='*'){//遇到字符*
            //连续只处理最后一个*，否则会超时
            while(p1+1<s1.length() && s1.charAt(p1+1)=='*') p1++;
            //如果p2指向不是能匹配字符，*匹配0个字符，p1后移
            if(!(Character.isLetterOrDigit(s2.charAt(p2)) || s2.charAt(p2)=='.')){
                return dfs(s1, s2, p1+1, p2);
            }
            //否则，p2指向字符可以匹配，*可以匹配0个，1个，或多个
            return dfs(s1, s2, p1+1, p2) || dfs(s1, s2, p1+1, p2+1) || dfs(s1, s2, p1, p2+1);
        }
        //其他不能匹配情况
        return false;
    }
}

HJ77 火车进站

题目地址：https://www.nowcoder.com/practice/97ba57c35e9f4749826dc3befaeae109?tpId=37&&tqId=21300&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：回溯

所有火车进出站都是由多步操作组合完成，而每步操作只有两种可能，要么只进站，要么只出站，然后继续递归，递归完成后恢复操作前的现场即可。

终止条件

  • 已全出站，out栈大小等于火车数组arr长度，注意new ArrayList<>(out)是从栈顶到栈底的元素顺序生成，而答案出站顺序是栈底到栈顶，结果转为字符串要从后往前遍历list，方便后续排序。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] arr = new int[n];
            for(int i=0; i<arr.length; i++) arr[i] = sc.nextInt();
            //双栈和Set集合
            Deque<Integer> in = new LinkedList<>();//站内的车
            Deque<Integer> out = new LinkedList<>();//已经出站的车
            List<String> res = new ArrayList<>();//结果集合
            //调用函数
            dfs(arr, 0, in, out, res);
            //排序
            Collections.sort(res);
            //输出结果
            for(String str : res){
                System.out.println(str);
            }
        }
    }
    
    //arr火车数组，k当前arr位置，in/out入站栈和出栈站，res结果
    private static void dfs(int[] arr, int k, Deque<Integer> in, Deque<Integer> out, List<String> res){
        //已全部出站
        if(out.size()==arr.length){
            //注意！！！new ArrayList<>()处理栈时是从栈顶到栈底的元素顺序
            List<Integer> list = new ArrayList<>(out);
            StringBuilder sb = new StringBuilder();
            for(int i=list.size()-1; i>=0; i--){
                if(i!=0) sb.append(list.get(i)+" ");
                else sb.append(list.get(i));
            }
            res.add(sb.toString());
            return;
        }
        
        if(!in.isEmpty()){//只出站不进站
            //选择出站
            int temp = in.peek();
            out.push(in.pop());
            dfs(arr, k, in, out, res);//出站不影响k
            //撤销选择，恢复现场
            in.push(temp);
            out.pop();
        }
        
        if(k!=arr.length){//只进站，不出站
            //选择进站
            in.push(arr[k]);
            dfs(arr, k+1, in, out, res);
            //撤销选择
            in.pop();
        }
    }
}

HJ89 24点运算

题目地址：https://www.nowcoder.com/practice/7e124483271e4c979a82eb2956544f9d?tpId=37&&tqId=21312&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：深度优先搜索

遍历四个数任一数开始运算，用StringBuilder拼接结果，递归所有未访问数以及四种运算，找不到就回溯。

代码

import java.util.*;

public class Main{
    private static final Map<Integer, String> map = new HashMap<Integer, String>(){{
        put(1, "A");
        put(2, "2");
        put(3, "3");
        put(4, "4");
        put(5, "5");
        put(6, "6");
        put(7, "7");
        put(8, "8");
        put(9, "9");
        //同一用一个字符存储，后面再替换
        put(10, "T");
        put(11, "J");
        put(12, "Q");
        put(13, "K");
    }};
    
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            String poker = sc.nextLine();
            if(poker.contains("joker") || poker.contains("JOKER")){//含有大小王
                System.out.println("ERROR");
            }else{//得到数字数组
                String[] str = poker.split("\\ ");
                int[] arr = new int[4];
                for(int i=0; i<4; i++){
                    String s = str[i];
                    if(s.equals("J")) arr[i] = 11;
                    else if(s.equals("Q")) arr[i]=12;
                    else if(s.equals("K")) arr[i] = 13;
                    else if(s.equals("A")) arr[i] = 1;
                    else arr[i] = Integer.parseInt(s);
                }
                //调用递归
                List<String> res = new ArrayList<>();
                StringBuilder sb = new StringBuilder();
                boolean[] isVisited = new boolean[4];
                //以某个数开始
                for(int i=0; i<4; i++){
                    //加入字符，置为已访问
                    sb.append(map.get(arr[i]));
                    isVisited[i]=true;
                    if(dfs(arr, 1, arr[i], isVisited, sb, res)) break;
                    //撤销字符，置为未访问
                    sb.deleteCharAt(sb.length()-1);
                    isVisited[i]=false;
                }
                //输出结果
                if(res.size()>0){
                    System.out.println(res.get(0));
                }else{
                    System.out.println("NONE");
                }
            }
        }
    }
    
    private static boolean dfs(int[] arr, int count, int val, boolean[] isVisited, StringBuilder sb, List<String> res){
        //终止条件
        if(val==24 && count==4){
            res.add(sb.toString().replaceAll("T", "10"));
            return true;
        }

        //遍历数组
        for(int i=0; i<arr.length; i++){
            if(!isVisited[i]){//未使用过
                // 选择arr[i]
                isVisited[i]=true;
                
                // +递归
                sb.append('+').append(map.get(arr[i]));
                if(dfs(arr, count+1, val+arr[i], isVisited, sb, res)) return true;
                sb.delete(sb.length()-2, sb.length());
                // -递归
                sb.append('-').append(map.get(arr[i]));
                if(dfs(arr, count+1, val-arr[i], isVisited, sb, res)) return true;
                sb.delete(sb.length()-2, sb.length());
                // *递归
                sb.append('*').append(map.get(arr[i]));
                if(dfs(arr, count+1, val*arr[i], isVisited, sb, res)) return true;
                sb.delete(sb.length()-2, sb.length());
                // /递归
                sb.append('/').append(map.get(arr[i]));
                if(dfs(arr, count+1, val/arr[i], isVisited, sb, res)) return true;
                sb.delete(sb.length()-2, sb.length());
                
                // 撤销选择arr[i]
                isVisited[i]=false;
            }
        }
        return false;
    }
}

HJ93 数组分组

题目地址：https://www.nowcoder.com/practice/9af744a3517440508dbeb297020aca86?tpId=37&&tqId=21316&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：深度优先搜索

先把三和五的倍数都挑出来，算好两边的和sum3和sum5，所有数总和为sum，不是3或5倍数的剩余的数放在集合中。
求出target = sum/2-sum3或者target=sum/2-sum5作为目标数，看list中找能不能凑出target。
在剩余集合中找target是一个dfs过程

  • 终止条件，用完集合数，返回target==0
  • 两种递归情况
    • 选择start位置，递归找新目标数target-list.get(start)
    • 不选择start位置，在start+1和以后位置找target

特例：sum不是2的倍数，不可等分成两份，直接输出false

代码

import java.util.*;
public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        while(in.hasNext()){
            //根据输入计算sum3，sum5和所有数总和sum，同时把不是5和3倍数的剩余数放入集合
            List<Integer> list = new LinkedList<>();
            int n = in.nextInt();
            int sum5=0, sum3=0, sum = 0;
            for (int i = 0; i < n; i++){
                int cur = in.nextInt();//输入
                if (cur % 5 == 0){//5倍数和
                    sum5 += cur;
                }else if (cur % 3 == 0){//3倍数和
                    sum3 += cur;
                }else{//剩余加入集合
                    list.add(cur);
                }
                sum += cur;//总和
            }
            
            //特例，总和不是2的倍数，不可分2份和相等的数字
            if(sum%2!=0) System.out.println("false");
            else{//否则，在剩余数中找目标数字
                int target = sum/2 - sum3;//也可以是sum/2-sum5
                System.out.println(helper(list, target, 0));
            }
        }
    }

    private static boolean helper(List<Integer> list, int target, int start){
        if (start == list.size()) return target == 0;//终止条件
        
        //选择start位置，递归找新目标数target-list.get(start)， 或者不选择start位置，在后面位置找target
        return helper(list, target-list.get(start), start+1) || helper(list, target, start+1);
    }
}

HJ102 字符统计

题目地址：https://www.nowcoder.com/practice/c1f9561de1e240099bdb904765da9ad0?tpId=37&&tqId=21325&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：List集合定制排序

由于Map集合定制排序比较困难，所以自己建立一个Node节点，存储key-value，将统计结果放入List集合，定制排序实现Comparator接口即可。

代码

import java.util.*;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            String s = sc.nextLine();
            //统计字符
            int[] arr = new int[36];
            for(int i=0; i<s.length(); i++){
                char c=s.charAt(i);
                if(c>='a' &&c<='z'){//字母
                    arr[c-'a']++;
                }else{//数字
                    arr[c-'0'+26]++;
                }
            }
            //统计结果存入List集合
            List<Node> list = new ArrayList<>();
            for(int i=0; i<36; i++){
                if(arr[i]!=0){
                    if(i<26){//字母
                        list.add(new Node((char)('a'+i), arr[i]));
                    }else{//数字
                        list.add(new Node((char)('0'+i-26), arr[i]));
                    }
                }
            }
            //排序
            Collections.sort(list, new Comparator<Node>(){
                public int compare(Node a, Node b){
                    if(a.v!=b.v){//根据值排序
                        return b.v-a.v;
                    }else{//值相等根据ASCII码排序
                        return a.k-b.k;
                    }
                }
            });
            //输出结果
            for(int i=0; i<list.size(); i++){
                System.out.print(list.get(i).k);
            }
            System.out.println();
        }
    }
}

class Node{
    char k;
    int v;
    
    public Node(char k, int v){
        this.k=k;
        this.v=v;
    }
}

HJ103 Redraiment的走法

题目地址：https://www.nowcoder.com/practice/24e6243b9f0446b081b1d6d32f2aa3aa?tpId=37&&tqId=21326&rp=1&ru=/ta/huawei&qru=/ta/huawei/question-ranking

方法一：动态规划

这道题目是最长递增子序列问题，不是将每个位置作为起点，而是作为终点，求出该位置的最大上升次数，同时根据终点不断更新最大值，得到最终结果。

  • 初始化，动态数组dp存储对应nums位置的桩的最大次数，默认为1，当前桩就是一步
  • 状态转移方程，扫描i之前的桩，如果比i小，则dp[i]=Math.max(dp[i], dp[j]+1)，意思是某个桩j比i小，踩j到i自然多了一步，取当前最大步数dp[i]和dp[j]+1最值更新

代码

import java.util.Scanner;
import java.util.Arrays;

public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            //处理输入
            int n = sc.nextInt();
            int[] arr = new int[n];
            for(int i=0; i<n; i++){
                arr[i] = sc.nextInt();
            }
            
            //动态规划数组，以i为终点的最长递增子序列
            int[] dp = new int[n];
            Arrays.fill(dp, 1);//初始化数组
            int max = 0;
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < i; j++) {//遍历i之前的位置
                    if (arr[i] > arr[j]) {//如果某个桩j比i小，踩在上面多一步，dp[i]和dp[j]+1取最值更新
                        dp[i] = Math.max(dp[i], dp[j] + 1);
                    }
                }
                max = Math.max(max, dp[i]);//更新max
            }
            System.out.println(max);
        }
    }
}

HJ107 求解立方根

方法一：二分法或牛顿迭代法

二分法比较复杂，需要分正负（用flag标识），以及[0,1]和[1,+∞]两部分进行二分查找。

牛顿迭代法是数学方法，设f(x)=x^3-y，求解f(x)=0时的解x，即为y的立方根。按牛顿迭代法的思想x(n+1) = x(n)-f(x_n)/f'(x_n) = x-(x^3-y)/3x^2 = (2x+y/x/x)*3。

import java.lang.Math;

public class Main{
    public static void main(String[] args){

    //牛顿迭代法，设f(x)=x^3-y，求解f(x)=0时的解x，即为y的立方根。
    //x(n+1) = x(n)-f(x_n)/f'(x_n) = x-(x^3-y)/3x^2 = (2x+y/x/x)*3
    private static double NewTon(double num){
        double x0 = num;
        double x1 = (2*x0+num/x0/x0)/3;
        while(Math.abs(x1-x0)>0.01){
            x0 = x1;
            x1 = (2*x0+num/x0/x0)/3;
        }
        return x1;
    }
    
    //二分法，小于等于1，在[num, 1]范围找
    private static double fun1(double num, boolean flag){
        double a=num;
        double b=1;
        double mid = (a+b)/2;
        while((b*b*b-a*a*a)>0.01){
            if(mid*mid*mid<num){
                a = mid;
            }else{
                b = mid;
            }
            mid = (a+b)/2;
        }
        if(flag) return -a;
        return a;

HJ108 求最小公倍数

方法一：辗转相除法

辗转相除法求两数最大公约数，最小公倍数=两数乘积/最大公约数

最大公约数，gbc(a, b)=gbc(b, a%b)，迭代或递归写法

代码

import java.util.Scanner;

//辗转相除法求最大公约数gbc(a,b)= gbc(b, a%b);，最小公倍数=两数乘积 / 最大公约数
public class Main{
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        while(sc.hasNext()){
            int a = sc.nextInt();
            int b = sc.nextInt();
            System.out.println((a*b)/gcb2(a, b));
        }
    }
    
    //递归写法gbc(a,b)= gbc(b, a%b)
    private static int gcb1(int a, int b){
        if(b==0) return a;
        
        //后面递归b大，a%b小，所以b先达到0
        return gcb1(b, a%b);
    }
    
    //迭代写法
    private static int gcb2(int a, int b){
        while(b>0){
            int temp = a%b;
            a = b;
            b = temp;
        }
        return a;
    }
}
